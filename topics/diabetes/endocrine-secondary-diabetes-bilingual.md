# 猫糖尿病内分泌继发性疾病 / Feline Diabetes Endocrine-Secondary Disease

## 本页回答的问题 / Question This Page Answers

何时应将猫糖尿病视为超出默认 2 型样疾病框架？

When should feline diabetes be treated as more than a default type-2-like disease frame?

## 核心要点 / Core Takeaway

2 型样猫糖尿病是默认框架，不是普遍解释。高生长激素血症、肢端肥大症和其他内分泌疾病需要命名的继发性糖尿病门控。

Type-2-like feline diabetes is the default frame, not the universal explanation. Hypersomatotropism, acromegaly, and other endocrinopathies need a named secondary-diabetes gate.

## 为什么此分支存在 / Why This Branch Exists

src-diabetes-013 支持结构性观点：许多猫符合 2 型样框架，而有意义的子集有其他糖尿病病因如高生长激素血症、肢端肥大症或高肾上腺皮质功能亢进。

src-diabetes-020 更新分支，将高生长激素血症诱导的糖尿病置于前景，作为可能难以管理的生长激素相关胰岛素抵抗问题。

src-diabetes-024 重要，因为疑似高生长激素血症排除影响如何解读胰岛素疗效证据。

## 当前门控 / Current Gate

当问题涉及以下内容时路由到本页：

- 困难的血糖控制 / difficult glycemic control
- 高胰岛素剂量需求 / high insulin dose requirements
- 多变的胰岛素反应 / variable insulin response
- 意外的非缓解 / unexpected non-remission
- 疑似肢端肥大症表型 / suspected acromegalic phenotype
- 排除疑似高生长激素血症病例的治疗研究 / treatment studies that exclude suspected hypersomatotropism cases

## 当前规则 / Current Rule

勿在未检查继发性内分泌疾病是否已被排除、纳入或忽略的情况下解释胰岛素或饮食反应。

Do not interpret insulin or diet response without checking whether secondary endocrine disease has been excluded, included, or ignored.
