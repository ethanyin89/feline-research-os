# Feline Diabetes Treatment Branch Map

## Question This Page Answers

How should treatment options be separated before the evidence is strong enough for ranking?

## Core Takeaway

The current diabetes treatment map has no safe single winner. The right structure is branch separation by role, evidence type, candidate selection, endpoint, and safety boundary.

## Branch Comparison

| Branch | Current Role | Strongest Safe Claim | Main Boundary |
|---|---|---|---|
| Diet architecture | foundational management | diet affects glycemic and insulin-requirement endpoints | not a universal prescription |
| Insulin / glargine U300 | core pharmacologic branch | promising short-term glargine U300 signal | no comparative superiority |
| SGLT2 inhibitors | U.S. FDA-regulated oral product branch | approved options for selected otherwise healthy insulin-naive cats | ketoacidosis risk and candidate selection |
| Alpha-glucosidase inhibitor adjunct | historical adjunct-plus-diet branch | combined intervention signal | independent drug effect not isolated |
| Incretin / GLP-1 / DPP-4 frontier | innovation branch | plausible metabolic mechanisms | not routine protocol authority |
| Remission | endpoint/control branch | real but evidence-fragile endpoint | no single predictor or protocol |

## Route By Question

- Diet composition or insulin requirement: [diet architecture](diet-architecture.md)
- Remission language: [remission boundaries](remission-boundaries.md)
- SGLT2 indication, candidate gate, or monitoring language: [SGLT2 label control](sglt2-label-control.md)
- Endocrine-secondary diabetes or difficult insulin response: [endocrine secondary diabetes](endocrine-secondary-diabetes.md)
- Pancreatitis or DKA complexity: [pancreatitis comorbidity](pancreatitis-comorbidity.md)

## Guardrail

Do not rank treatments by superiority from this layer. Do not promote oral route as inherently better. Do not compare Bexacat and Senvelgo as winners. Do not write owner-facing clinical instructions from the current compiled layer alone.
