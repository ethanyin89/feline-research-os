# 猫糖尿病缓解边界 / Feline Diabetes Remission Boundaries

## 证据深度警告 / Evidence-Depth Caveat

本页现在基于深度提取的论文卡片层，但仍不是全文压缩或决策级缓解预测或治疗指导页面。

## 本页回答的问题 / Question This Page Answers

关于猫糖尿病缓解，知识库今天能安全地说什么？

What can the vault safely say about feline diabetic remission today?

## 核心要点 / Core Takeaway

猫糖尿病缓解是真实的，但当前种子证据不支持单一预测因子、饮食、胰岛素类型或方案作为普遍缓解关键。

Feline diabetic remission is real, but the current seed evidence does not support a single predictor, diet, insulin type, or protocol as the universal remission key.

## 允许的含义 / Allowed Meanings

缓解可用作：

- 猫糖尿病的核心终点 / a core endpoint in feline diabetes
- 与短期血糖改善分开追踪胰岛素独立性的理由 / a reason to track insulin independence separately from short-term glycemic improvement
- 当证据和监测明确时的合法治疗目标 / a legitimate treatment goal when evidence and monitoring are explicit
- 与饮食、胰岛素、肥胖、胰腺炎和内分泌继发性疾病相互作用的边界 / a boundary that interacts with diet, insulin, obesity, pancreatitis, and endocrine-secondary disease

## 尚不允许 / Not Allowed Yet

缓解不应用于声称：

- 一种饮食配方可靠地导致缓解 / one diet formulation reliably causes remission
- 一种胰岛素类型可靠地预测缓解 / one insulin type reliably predicts remission
- 不检查定义和设计就可以跨研究比较缓解率 / remission rates can be compared across studies without checking definitions and design
- 缓解潜力消除慢性 β 细胞衰竭或继发性内分泌疾病 / remission potential erases chronic beta-cell failure or secondary endocrine disease
- 非胰岛素依赖自动等同于完全定义的缓解终点 / non-insulin-dependence is automatically identical to a fully defined remission endpoint

## 标签规则 / Labeling Rule

当另一个糖尿病页面提到缓解时，将声明标记为以下之一：

- `终点 endpoint`: 缓解作为结果被追踪
- `信号 signal`: 研究报告胰岛素独立性或缓解样结果
- `边界 boundary`: 证据质量阻止方案排名
- `未建立 not established`: 提出预测因子或层次但尚未有来源支持

## 护栏 / Guardrail

缓解应可见，但不应主导整个疾病模型。证据质量阻止简单的预测因子或方案排名。

Remission should be visible, but it should not dominate the whole disease model. Evidence quality blocks simple predictor or protocol ranking.
