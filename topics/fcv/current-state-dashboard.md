# Feline FCV Current State Dashboard

## Current Module State

FCV is now a mature disease module with 24/24 source cards deep-extracted and core topic pages recompiled against the complete source-card layer. The module covers broad review/control (`src-fcv-001`, `src-fcv-002`, `src-fcv-004`, `src-fcv-007`, `src-fcv-009`, `src-fcv-015`), epidemiology/recognition (`src-fcv-005`, `src-fcv-006`, `src-fcv-021`, `src-fcv-024`), vaccine/immunity (`src-fcv-003`, `src-fcv-010`, `src-fcv-011`, `src-fcv-012`, `src-fcv-013`, `src-fcv-017`, `src-fcv-022`, `src-fcv-024`), therapy (`src-fcv-008`, `src-fcv-014`, `src-fcv-018`), and mechanism/extension (`src-fcv-015`, `src-fcv-016`, `src-fcv-020`, `src-fcv-021`, `src-fcv-023`).

## Source-Family Read

- guideline and broad review sources should control disease shell, control architecture, and mechanism compression
- original studies should control endpoint ownership, challenge detail, field-recognition detail, and branch-specific mechanism signals
- commentary/context material should remain low and should not control FCV hierarchy, treatment, or ranking language

## Current Claim-Fit Warnings

- do not flatten `src-fcv-003`, `src-fcv-011`, `src-fcv-017`, and `src-fcv-022` into one generic `vaccine success` bucket
- do not let in-vitro breadth, challenge protection, cellular immunity, and chronic-carrier control drift into one protection claim
- do not let `src-fcv-019` or other context-style material control disease architecture or treatment prose
