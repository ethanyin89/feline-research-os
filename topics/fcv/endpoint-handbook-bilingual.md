## 核心要点 / Core Takeaway

FCV 终点语言目前应保持在中和广度、挑战保护、急性临床严重性、病毒排毒、携带者状态持久性和血清学抵抗预测之间分离。

FCV endpoint language should stay separated between neutralisation breadth, challenge protection, acute clinical severity, viral shedding, carrier-state persistence, and serologic resistance prediction.

## 当前安全解读 / Current Safe Read

- 中和广度不等于灭菌免疫 / Neutralisation breadth ≠ sterilising immunity
- 挑战保护不等于现场控制 / Challenge protection ≠ field control
- 抗体阳性不等于普遍长期保护 / Antibody positivity ≠ universal long-term protection
- 急性疾病减少不能消除慢性携带 / Reduced acute disease does not erase chronic carriage

## 工作终点层级 / Working Endpoint Hierarchy

最安全的首次 FCV 终点顺序：
1. 急性临床疾病减少 / Acute clinical-disease reduction
2. 异源挑战保护 / Heterologous challenge protection
3. 交叉中和广度 / Cross-neutralisation breadth
4. 细胞交叉免疫 / Cellular cross-immunity
5. 排毒和慢性携带者持久性 / Shedding and chronic-carrier persistence
6. 血清学抵抗预测 / Serologic resistance prediction

此顺序有助于防止血清学、中和和疫苗研究被压缩为单一保护指标。

## 护栏 / Guardrail

不要在滴度、挑战读数、平台疫苗发现和携带结果之间建立过早的 FCV 数字排行榜。主要终点家族已稳定，但分支仍需更密集的现场有效性和标签/监管证据才能成为更完整的终点手册。

Do not build a premature FCV numeric leaderboard. Major endpoint families are now stabilized, but the branch still needs denser field-effectiveness and label/regulatory evidence.
