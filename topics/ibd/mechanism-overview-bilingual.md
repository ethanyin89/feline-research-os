## Current Mechanism Map / 当前机制地图

### Framing Layer / 框架层

**EN**
- `chronic enteropathy` is the broader framing layer
- feline IBD should remain inside that wider operational frame
- the mechanism page should not collapse chronic enteropathy into one narrow idiopathic-IBD story too early

**ZH**
- `chronic enteropathy` 是更宽的框架层
- 猫 IBD 应继续稳定地放在这个更宽的操作框架里
- mechanism 页不应过早把 chronic enteropathy 压成一个狭窄的 idiopathic-IBD 故事

### Core Inflammatory Layer / 核心炎症层

**EN**
- `mucosal inflammation` is the central operational disease process
- idiopathic-IBD immunopathology now has its own depth layer
- this depth improves explanation more than it changes frontline workup order

**ZH**
- `mucosal inflammation` 是核心操作性疾病过程
- idiopathic-IBD immunopathology 现在已经有了自己的深度层
- 这层深度更多是在增强解释，而不是改写 frontline workup order

### Microbiota-Dysbiosis Layer / 微生物群失衡层

**EN**
- `microbiota dysbiosis` is a plausible recurrent mechanism branch, but not yet a lead diagnostic endpoint
- current microbiota evidence already suggests that dysbiosis can touch the `IBD versus small-cell lymphoma` boundary, not only generic inflammation
- older FISH-based work anchors the pre-omics side of this branch and keeps newer frontier work from floating without a spine
- cross-species microbiota review material widens context, but does not replace feline-primary anchors

**ZH**
- `microbiota dysbiosis` 是一个可信的重复出现机制分支，但还不是 lead diagnostic endpoint
- 当前 microbiota 证据已经提示，这条失衡分支会触碰 `IBD versus small-cell lymphoma` 的边界，而不只是一般性炎症
- 更早的 FISH-based 工作锚定了这条分支的 pre-omics 一侧，也避免较新的 frontier work 漂浮失根
- 跨物种 microbiota 综述可以拉宽语境，但不能替代 feline-primary anchors

### Lower Marker-Depth Layer / 更低一层的 marker-depth

**EN**
- current molecular and tissue-marker evidence suggests that much of the lower-layer signal is shared-response biology rather than clean disease-class separation
- these signals deepen mechanism more than they deliver operational class identity

**ZH**
- 当前分子和组织标志物证据提示，这一层许多信号更像 shared-response biology，而不是干净的 disease-class separation
- 这些信号更多是在加深机制解释，而不是提供操作性的类别身份

### Structural Chronicity And Remodeling Layer / 结构性慢性化与重塑层

**EN**
- `fibrosis and remodeling` now belong to a real structural chronicity branch rather than a side note
- fibrosis should sit above support-marker burden signals in the structural disease map
- extension remodeling phenotypes such as `eosinophilic sclerosing fibroplasia` can widen the remodeling map without being collapsed into core idiopathic IBD

**ZH**
- `fibrosis and remodeling` 现在已经属于一个真实的结构性慢性化分支，而不是旁注
- 在结构性疾病地图里，fibrosis 应放在 support-marker burden signals 之上
- `eosinophilic sclerosing fibroplasia` 这类 extension remodeling phenotypes 可以拉宽重塑地图，但不应被压进 core idiopathic IBD

### Boundary-Adjacent Layer / 邻接边界层

**EN**
- the `IBD versus small-cell lymphoma` boundary must stay adjacent to mechanism
- it should not be treated only as pathology administration or report wording

**ZH**
- `IBD versus small-cell lymphoma` 的边界必须继续贴近机制层
- 它不应只被当成 pathology administration 或报告措辞问题

## What We Can Defend / 当前可以稳妥主张的内容

**EN**
- feline IBD mechanism is best modeled as layered rather than single-cause
- chronic enteropathy remains the correct outer frame
- dysbiosis is mechanistically important without becoming frontline diagnostic authority
- fibrosis deserves explicit structural-chronicity status

**ZH**
- 猫 IBD 的机制最适合被建模成分层结构，而不是 single-cause 故事
- chronic enteropathy 仍是正确的外层框架

- fibrosis 值得被明确放在 structural-chronicity 的位置上

## What We Should Not Overstate / 当前不应说过头的内容

- molecular-expression papers should not outrank histology and workup utility
- extension branches should not overwrite the core IBD frame

**ZH**
- microbiota correlation 不等于 disease-cause certainty
