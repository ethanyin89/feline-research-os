# 猫慢性肾病 SDMA 定位 / Feline CKD SDMA Positioning

## 本页回答的问题 / Question This Page Answers

关于猫 CKD 中的 SDMA，本知识库目前能说什么，不能说什么？

What can this vault currently say, and not say, about SDMA in feline CKD?

## 当前结论 / Current Conclusions

### 引用事实 / Quoted Fact

- 诊断和分期综述将 GFR 视为检测早期肾功能障碍的理想指标，但指出了实际常规使用的局限性
- ISFM 指南指出，老年猫每年或每半年连续评估血清肌酐或 SDMA 及 USG 可能有助于更早或更确定地诊断
- 同一指南指出，SDMA 似乎比肌酐对早期 CKD 检测具有更高的敏感性，且似乎不受肌肉质量影响
- 该指南还指出，SDMA 目前不能作为 CKD 的单一筛查试验推荐，但可能支持诊断或分期，特别是在肌肉明显流失的猫中
- 2022 年生物标志物综述指出，血清肌酐在早期 CKD 检测方面存在重要局限性，SDMA 作为新型 GFR 生物标志物被引入用于猫的早期检测
- 同一综述指出 SDMA 的特异性数据仍然有限

### 来源支持的结论 / Source-Supported Conclusion

- 在当前知识库中，SDMA 属于终点图谱中的辅助性早期检测和分期工具
- SDMA 目前应位于连续肌酐解释加 USG 这一经典实用核心之下，而非之上
- SDMA 目前最强的使用场景是减少对单一肌酐思维的过度依赖，特别是在肌肉流失使解释复杂化的情况下
- 当前语料库支持 SDMA 作为支持性标志物，而非替代性标志物
- 新的生物标志物综述降低了关于 SDMA 是否重要的不确定性，但并未消除关于它在常规工作流程中应如何加权的不确定性
- 代谢组学加机器学习论文加强了 SDMA 作为更广泛前沿领域中一个成分的角色，但削弱了任何单独 SDMA 捷径叙事

### LLM 推理 / LLM Inference

- 当前瓶颈不再是 SDMA 是否重要——它确实重要
- 真正未解决的问题是相对于肌酐趋势、USG 和更广泛的连续监测逻辑，它应该如何加权

## 证据图谱 / Evidence Map

- `src-ckd-002`: 诊断/分期框架，GFR 理想参考逻辑
- `src-ckd-004`: 知识库中最强的 SDMA 表述
- `src-ckd-018`: 代谢组学加 ML 前沿论文，削弱任何单独 SDMA 捷径叙事
- `src-ckd-024`: 生物标志物综述框架，支持 SDMA 相关性同时保持谨慎

## 可以安全表述的内容 / What We Can Say Safely

1. SDMA 与猫 CKD 的早期识别相关
2. 当肌酐解释可能因肌肉质量减少而失真时，SDMA 特别有用
3. SDMA 应在连续和多标志物工作流程中解释
4. SDMA 不能支持单一测试筛查叙事
5. 在当前前沿文献中，SDMA 更适合被解读为更广泛增强层中的一个组成部分，而非单一主导答案

## 目前不应表述的内容 / What We Should Not Say Yet

1. SDMA 取代肌酐
2. SDMA 取代 USG
3. SDMA 已经是常规实践中最好的单一早期 CKD 标志物
4. 当前知识库有足够证据将 SDMA 与所有传统标志物进行明确排名

## V1 工作定位 / Working Position In V1

| 标志物 Marker | 当前 V1 定位 | 原因 Why |
|---|---|---|
| 肌酐 Creatinine | 核心操作锚点 | 连续变化在当前语料库中仍然实用且核心 |
| USG | 核心操作锚点 | 肾性氮质血症和 CKD 确认的必要背景 |
| SDMA | 辅助性暂定核心 | 有助于早期识别和肌肉质量边缘案例，但不是独立捷径 |
| GFR | 参考标准概念 | 理论上理想，常规使用有限 |

## 操作规则 / Operational Rule

如果 CKD 笔记、备忘录或输出提到 SDMA，通常还应提到以下至少一项：

- 连续肌酐趋势 / serial creatinine trend
- USG
- 老年猫监测逻辑 / older-cat surveillance logic
- 肌肉流失背景 / muscle-loss context

这将 SDMA 保持在正确的解释框架内。

## 当前差距 / Current Gaps

- 不应从当前语料库假设稳定的 SDMA 阈值或排名逻辑
- 尚无代谢组学或机器学习增强的常规诊所实施来源
- 尚不能在 SDMA、肌酐、USG 和前沿面板方法之间声称明确的排名顺序

## 护栏 / Guardrail

SDMA 是辅助标志物，不是替代标志物。在全语料库审查解决相对权重问题之前，不要声称 SDMA 优于或取代传统核心工作流程。

SDMA is an adjunctive marker, not a replacement marker. Do not claim SDMA superiority over or replacement of traditional core workflows until full-corpus review resolves relative weighting.
