# 猫慢性肾病影像学背景 / Feline CKD Imaging Context

## 本页回答的问题 / Question This Page Answers

影像学，特别是超声检查，在猫 CKD 检查中应扮演什么角色，同时不被过度推广？

What role should imaging, especially ultrasonography, play inside feline CKD workup without being overpromoted?

## 当前结论 / Current Conclusions

### 引用事实 / Quoted Fact

- ISFM 指南将诊断影像学作为怀疑 CKD 时最低常规数据库的一部分
- 超声检查综述指出超声是猫肾脏的参考影像模式
- 超声检查比 X 线更准确地评估猫肾脏，且在普通实践中比 CT 或 MRI 更易获得
- 超声可良好地可视化肾脏大小、形状和内部结构
- 局灶或多灶性肾脏疾病比弥漫性肾脏变化更容易在超声上识别
- B 型超声在区分良恶性局灶病变方面有限

### 来源支持的结论 / Source-Supported Conclusion

- 影像学属于 CKD 检查层，而非第一波终点层
- 超声最好被理解为结构背景和肾脏鉴别工具
- 当前知识库中影像学的最强作用是改善检查完整性和现实性，而非取代生化或尿液 CKD 解释
- 弥漫性 CKD 在超声上仍比局灶性肾脏疾病更难定义，这对过度声称设置了明确边界

### LLM 推理 / LLM Inference

- 知识库的实用规则是：超声帮助回答"我们正在处理什么结构性肾脏背景？"比回答"这只猫确实有早期弥漫性 CKD 吗？"更有效

## 影像学矩阵 / Imaging Matrix

| 影像学角色 Imaging Role | 帮助什么 | 不解决什么 | 当前最佳解读 |
|---|---|---|---|
| 结构背景 | 肾脏大小、形状、结构 | 单独精确的弥漫性 CKD 分类 | 有用且实用 |
| 检查完整性 | 当怀疑 CKD 时添加非实验室肾脏背景 | 取代肌酐、USG、UPC、SBP 逻辑 | 应与核心检查标志物并列，而非在其之上 |
| 鉴别支持 | 帮助识别局灶或多灶性肾脏异常 | 仅用 B 型清晰的良性与恶性病变定型 | 有界但临床有意义 |

## 本页清楚表述的内容 / What This Page Says Clearly

1. 影像学是良好的猫 CKD 检查现实性的一部分
2. 超声是猫肾脏的实用参考影像模式
3. 超声在结构背景和鉴别支持方面比单独解决弥漫性 CKD 解释更有用
4. 知识库不应将影像学背景与第一波 CKD 终点状态混淆

## 不应过度陈述的内容 / What We Should Not Overstate

- 超声是核心 CKD 终点
- 弥漫性猫 CKD 可以单独通过超声可靠地解决
- 仅 B 型超声解决病变类型

## 护栏 / Guardrail

影像学是检查背景工具，不是终点替代。超声帮助结构评估和鉴别，但不解决弥漫性 CKD 解释。

Imaging is a workup-context tool, not an endpoint replacement. Ultrasound helps with structural assessment and differential support, but does not solve diffuse CKD interpretation.
