## Core Takeaway / 核心要点

CKD recognition should not wait for obvious late-stage illness. The best-supported approach is deliberate surveillance of older or high-risk cats. Owner-observed changes (polyuria, polydipsia) should be treated as meaningful recognition prompts. Hyperthyroidism comorbidity complicates interpretation and requires active post-treatment renal reassessment.

CKD 的识别不应等到出现明显的晚期疾病症状。目前获得最强支持的方法是对老年猫或高风险猫进行刻意的监测。宠物主人观察到的变化（多尿、多饮）应被视为有意义的识别提示。甲亢共病会使指标解读复杂化，在治疗后需要进行积极的肾脏重新评估。

## Recognition Architecture / 识别架构

### Core Recognition Triggers / 核心识别触发因素

**Age-based surveillance / 基于年龄的监测:** Older cats (>7 years) should ideally have health checks every 6 months and selected diagnostic testing performed at least annually. Age is the strongest practical population-level trigger.
基于年龄的监测：老年猫（>7 岁）理想情况下应每 6 个月进行一次健康检查，并至少每年进行一次选定的诊断性检测。年龄是目前最有效的实际人群级触发因素。

**High-risk contexts / 高风险背景:** CKD should be considered in cats with:
高风险背景：在患有以下疾病的猫中应考虑 CKD：
- Systemic hypertension / 全身性高血压
- Hyperthyroidism / 甲状腺功能亢进（甲亢）
- Cardiovascular disease / 心血管疾病
- Ureterolithiasis / 输尿管结石
- Urinary tract infection / 尿路感染（UTI）
- Retrovirus infection / 逆转录病毒感染

**Lead sources / 主要文献源:** `src-ckd-001`, `src-ckd-004`, `src-ckd-005`

### Owner-Observed Recognition / 宠物主人观察识别

Owner-observed polydipsia and polyuria in the year before diagnosis are more common in CKD cases than controls. This suggests earlier recognition should often have been possible.
在诊断前一年内，宠物主人观察到的多饮和多尿在 CKD 病例中比对照组更为常见。这表明更早的识别通常是完全可能的。

**Key finding from `src-ckd-012` / `src-ckd-012` 的关键发现:** Earlier CKD diagnosis should have been possible in most cases. Recognition fails partly because observable daily-life changes are not converted into timely veterinary workup.
`src-ckd-012` 的关键发现：在大多数病例中，更早诊断 CKD 本应是可能的。识别失败的部分原因在于日常生活中可观察到的变化未能转化为及时的兽医临床检查。

**Current safe read / 当前安全解读:**
- Polyuria and polydipsia belong in recognition logic, not as late-stage trivia
- Owner-observed changes can precede formal diagnosis
- Recognition failures are partly behavioral and workflow-related, not only biomarker limitations

当前安全解读：
- 多尿与多饮应纳入识别逻辑，而不应仅视为晚期无足轻重的琐碎临床表现
- 主人观察到的改变可能在正式诊断之前出现
- 识别失败部分与行为习惯和诊疗流程有关，而仅是生物标志物的局限性

**Lead sources / 主要文献源:** `src-ckd-012`

### Diagnostic Workup / 诊断性检查

**Practical diagnosis / 实际诊断:** In practice, feline CKD is commonly diagnosed by:
实际诊断：在临床实践中，猫 CKD 通常通过以下方式确诊：
- Increased serum creatinine concentration >140 µmol/l (>1.6 mg/dl) / 血清肌酐浓度持续升高 >140 µmol/l (>1.6 mg/dl)
- Inadequately concentrated urine (USG <1.035) / 尿液浓缩不足（尿比重 USG <1.035）
- Persistence of these changes over time / 这些变化随时间推移具有持续性

**Minimum routine database / 最低限度常规检查库:**
- Urinalysis with UPCR / 尿液分析及尿蛋白肌酐比（UPCR）
- Serum biochemistry / 血清生化
- Haematology / 血常规
- Systolic blood pressure / 收缩压（SBP）
- Diagnostic imaging / 诊断性影像学

**Early dysfunction detection / 早期肾功能障碍检测:** A persistent increase in serum creatinine >15% from baseline may indicate reduced renal function even before overt azotaemia.
早期功能障碍检测：血清肌酐比基线值持续升高 >15% 可能表明肾功能已有所下降，即使尚未出现明显的氮质血症。

**SDMA positioning / SDMA 定位:** SDMA appears more sensitive than creatinine for early CKD detection and is less affected by muscle mass, but cannot currently be recommended as a single screening test.
SDMA 的定位：SDMA 在早期 CKD 检测中似乎比肌酐更为敏感，且受肌肉量影响较小，但目前不能推荐将其作为唯一的筛查手段。

**Lead sources / 主要文献源:** `src-ckd-004`

### Hyperthyroidism Comorbidity / 甲状腺功能亢进共病

Hyperthyroidism and CKD are both common in older cats and may be concurrent. Hyperthyroidism can mask CKD by various mechanisms.
甲状腺功能亢进（甲亢）和 CKD 在老年猫中都很常见，且可能同时发生。甲亢可通过多种机制掩盖 CKD。

**Key interpretation issues / 关键解读问题:**
- Concurrent hyperthyroidism requires careful monitoring of GFR biomarkers / 同时患有甲亢时，需要仔细监测 GFR（肾小球滤过率）生物标志物
- Restoration of euthyroidism may unmask or worsen azotaemia / 恢复正常的甲状腺功能可能会暴露或加重氮质血症
- Iatrogenic hypothyroidism increases azotaemia risk / 医源性甲状腺功能减退（甲减）会增加氮质血症的风险
- Older cats with vomiting, weight loss, PU/PD, anorexia, or sarcopenia should prompt consideration of both diseases / 伴有呕吐、体重减轻、多尿/多饮、厌食或肌肉减少症的老年猫，应同时考虑这两种疾病

**Lead sources / 主要文献源:** `src-ckd-019`

### Pre-Anaesthetic Screening / 麻醉前筛查

Pre-anaesthetic evaluation is a practical checkpoint that may catch occult disease earlier. Not equivalent to dedicated longitudinal surveillance, but a useful workflow trigger.
麻醉前评估是一个实用的检查节点，可以更早地发现隐匿性疾病。这虽不等于专门的长期纵向监测，但却是一个非常有用的流程触发点。

**Lead sources / 主要文献源:** `src-ckd-005`

## Guardrail / 护栏

Do not reduce recognition to a single test or single sign. The safe architecture is:
不要将识别简化为单一测试或单一临床症状。安全的评估架构为：

1. **Population-level trigger:** Age and comorbidity-based surveillance
   **人群级触发因素：** 基于年龄和共病的日常监测
2. **Symptom-level prompt:** Owner-observed PU/PD as workup escalation cue
   **症状级提示：** 以主人观察到的多尿/多饮作为启动深入检查的信号
3. **Diagnostic confirmation:** Creatinine + USG + persistence, plus minimum workup database
   **确诊验证：** 肌酐 + 尿比重 + 异常持续性，并配合最低限度检查库
4. **Comorbidity interpretation:** Hyperthyroidism complicates biomarker reading; requires active reassessment
   **共病解读：** 甲亢会使生物标志物的解读复杂化；治疗后需要积极重新评估

## What The Module Can Say Safely / 本模块可以安全宣称的内容

- Late diagnosis is a central strategic bottleneck in feline CKD
  晚期诊断是猫 CKD 临床管理中的核心战略瓶颈
- Targeted high-risk screening is more defensible than universal breakthrough promises
  针对高风险群体的定向筛查，比宣称能实现通用性早期突破的承诺更为可靠
- Owner-observed polyuria and polydipsia often precede formal diagnosis
  主人观察到的多尿与多饮通常在正式确诊前就已经出现
- Hyperthyroidism can mask CKD and complicates post-treatment biomarker interpretation
  甲亢可掩盖 CKD，并会使治疗后的生物标志物解读复杂化
- Age-based surveillance with annual testing is the strongest supported recognition approach
  基于年龄、配合年度检测的常规监测是获得最强证据支持的识别方法
- SDMA is an adjunctive early-detection aid, not a standalone screening replacement
  SDMA 是早期检测的辅助手段，不能作为独立的筛查替代方案
