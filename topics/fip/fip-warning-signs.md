# FIP怎么识别？/ How to Recognize FIP

## 最重要的警示信号

如果你的猫出现以下情况，应该考虑FIP的可能性：

### 🚨 立即就医的情况

| 症状组合 | 说明 |
|----------|------|
| 肚子变大 + 精神差 | 腹水是湿性FIP的典型表现 |
| 反复发烧 + 消瘦 | 吃退烧药也不管用的发烧 |
| 眼睛异常 + 发烧 | 眼睛浑浊、颜色改变 |
| 走路不稳 + 发烧 | 神经型FIP的表现 |

## 分步骤判断

### 第一步：猫咪是高风险群体吗？

| 问题 | 高风险答案 |
|------|-----------|
| 猫咪多大？ | 1岁以下 |
| 来自哪里？ | 猫舍、收容所、多猫家庭 |
| 最近有压力吗？ | 刚搬家、刚到新家、刚做手术 |

**如果符合多项 → 警惕性提高**

### 第二步：有没有这些早期症状？

| 早期症状 | 具体表现 |
|----------|----------|
| 食欲下降 | 以前爱吃，现在不想吃 |
| 精神变差 | 不爱玩，总是睡觉 |
| 体重下降 | 明显变瘦 |
| 发烧 | 摸起来比平时热 |

**如果有2项以上 → 尽快看医生**

### 第三步：有没有这些典型症状？

#### 湿性FIP（更容易发现）

| 症状 | 怎么看 |
|------|--------|
| 腹部膨大 | 肚子像气球一样鼓起来，摸起来有液体感 |
| 呼吸急促 | 胸腔积液导致呼吸困难 |
| 黄疸 | 眼白、耳朵内侧、牙龈发黄 |

#### 干性FIP（更难发现）

| 症状 | 怎么看 |
|------|--------|
| 眼睛问题 | 瞳孔颜色改变、眼睛浑浊、怕光 |
| 走路异常 | 走路摇晃、后腿无力、站不稳 |
| 行为异常 | 性格突变、不认人、躲起来 |
| 长期消瘦 | 吃东西但越来越瘦 |

## 容易和FIP混淆的情况

| 症状 | 也可能是 | 区别 |
|------|----------|------|
| 腹水 | 心脏病、肝病、肿瘤 | FIP腹水通常是黄色粘稠的 |
| 发烧 | 感染、炎症 | FIP发烧对抗生素无效 |
| 消瘦 | 寄生虫、甲亢、癌症 | FIP伴随持续发烧 |
| 眼睛问题 | 眼科疾病 | FIP伴随全身症状 |

**重要：** 确诊FIP需要医生综合判断，不能仅靠单一症状。

## 什么时候该去医院？

### 必须立即去

- 呼吸困难
- 肚子突然变大
- 抽搐
- 完全不吃东西超过24小时

### 尽快预约

- 反复发烧超过3天
- 明显消瘦
- 眼睛出现异常
- 走路不稳

### 普通观察

- 偶尔食欲不好
- 偶尔精神不佳
- 轻微腹泻

## 去医院前准备什么

1. **记录症状** — 什么时候开始的？变化过程？
2. **拍照/录像** — 异常行为、眼睛变化、肚子大小
3. **带上病历** — 之前的检查结果、疫苗记录
4. **告诉医生** — 猫咪来源、家里其他猫的情况

## 本页边界

这个页面帮助你判断是否需要怀疑FIP。

**本页不能：**
- 替代医生诊断
- 提供用药建议
- 确认是否是FIP

**确诊需要：** 医生结合病史、体检、血液检查、影像检查、甚至腹水检查来综合判断。

如果怀疑FIP，请尽快就医。现在FIP有药可治，早发现早治疗效果更好。
