## Support Order / 支持顺序

**EN**
- clinicopathology leads
- acute-phase, immunoglobulin, and older serology stay in background support
- mutation-related assays strengthen after suspicion is already real
- CSF support becomes central only after neurologic branch shift

**ZH**
- clinicopathology 先主导
- acute-phase、immunoglobulin 和较旧的 serology 保持在背景支持层
- mutation-related assays 要在 suspicion 已经成立之后再进入 strengthening
- CSF support 只有在 neurologic branch shift 之后才会变成中心

## Current Endpoint Map / 当前终点地图

### Clinicopathology-Led Support / 由临床病理主导的支持层

**EN**
- clinicopathology and disease-form patterning belong to the lead operational branch
- suspicion-building should remain above molecular strengthening
- this branch raises recognition strength without becoming certainty by itself

**ZH**
- clinicopathology 和 disease-form patterning 属于主导的 operational branch
- suspicion-building 应继续放在 molecular strengthening 之上
- 这一分支会提高 recognition strength，但本身不能变成 certainty

### Inflammatory And Historical Laboratory Support / 炎症与历史实验室支持层

**EN**
- acute phase proteins and immunoglobulin patterns belong to bounded support
- older serology belongs to background context, not modern leadership
- these layers can shape interpretation without becoming front-door authority

**ZH**
- acute phase proteins 和 immunoglobulin patterns 属于有边界的 support
- 较旧的 serology 属于背景语境，而不是现代主导层
- 这些层可以帮助解释，但不能变成 front-door authority

### Mutation And Molecular Support / 突变与分子支持层

**EN**
- mutation-related assays belong to bounded diagnostic strengthening
- utility and limitation must stay together
- this layer should remain below composite-support architecture

**ZH**
- mutation-related assays 属于有边界的 diagnostic strengthening
- utility 和 limitation 必须放在一起
- 这一层应继续放在 composite-support architecture 之下

### Neurologic-Extension Support / 神经系统延伸支持层

**EN**
- CSF viral detection belongs to a specialized neurologic branch
- subgroup usefulness can be high after the workup has already narrowed
- this branch should not become general FIP endpoint leadership

**ZH**
- CSF viral detection 属于专门的神经系统分支
- 当 workup 已经缩窄后，它在 subgroup 中的价值可以很高
- 这一分支不应变成 general FIP endpoint leadership

### Treatment-Follow-Up Branch / 治疗随访分支

**EN**
- response, remission, relapse, and survival belong more naturally to a follow-up architecture
- treatment follow-up should stay distinct from first-pass suspicion logic
- outcome interpretation and diagnosis support should not be flattened together

**ZH**
- response、remission、relapse 和 survival 更自然地属于 follow-up architecture
- treatment follow-up 应继续和 first-pass suspicion logic 分开
- outcome interpretation 和 diagnosis support 不应被压平到一起

## What We Can Defend / 当前可以稳妥主张的内容

**EN**
- FIP endpoint logic should be modeled as layered support rather than one-marker hierarchy
- clinicopathology currently leads the most operational branch
- support order is clearer than any final assay leaderboard
- mutation support is meaningful but bounded
- neurologic-extension support matters without replacing general workup

**ZH**
- FIP 的 endpoint logic 应被建模成 layered support，而不是 one-marker hierarchy
- clinicopathology 目前仍然主导最 operational 的分支
- support order 目前比任何最终 assay leaderboard 都更清楚
- mutation support 是有意义的，但有边界
- neurologic-extension support 很重要，但不能替代 general workup

## What We Should Not Overstate / 当前不应说过头的内容

**EN**
- no single endpoint currently settles FIP by itself
- the vault still does not support a clean numeric assay ranking
- mutation support is not certainty language
- serology is not modern first-wave authority
- CSF support is not ordinary workup leadership

**ZH**
- 目前没有任何单一 endpoint 能独自坐实 FIP
- 这个 vault 目前仍不支持一个干净的 numeric assay ranking
- mutation support 不是 certainty language
- serology 不是现代 first-wave authority
- CSF support 不是 ordinary workup leadership
