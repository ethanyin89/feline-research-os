## Current Mechanism Map / 当前机制地图

### Structural Phenotype Layer / 结构表型层

**EN**
- HCM should still be read first as a structural phenotype disease.
- echocardiographic phenotype remains the visible front door of the mechanism map.
- phenotype heterogeneity matters and should not be flattened into one wall-thickening pattern.

**ZH**
- HCM 仍应首先被理解成结构表型疾病。
- echocardiographic phenotype 仍是机制地图的可见前门。
- 表型异质性很重要，不应被压成一种 wall-thickening pattern。

### Genetic Pressure Layer / 遗传压力层

**EN**
- genetics belongs near the top layer of HCM explanation
- genotype pressure should stay stratified, not binary
- dosage-sensitive severity and age-related penetrance are already meaningful in the current module

**ZH**
- genetics 属于 HCM 解释层的顶层分支之一
- genotype pressure 应继续保持分层，而不是二元化
- dosage-sensitive severity 和 age-related penetrance 在当前模块里已经具有实际意义

### Remodeling Layer / 重塑层

**EN**
- remodeling is not a passive after-effect
- fibrosis, macrophage-rich interstitial change, and profibrotic signaling deepen the mechanism spine
- the disease should not be modeled as phenotype without tissue-process depth

**ZH**
- remodeling 不是被动后果
- fibrosis、macrophage-rich interstitial change 和 profibrotic signaling 会加深机制主干
- 这个疾病不应被建模成没有 tissue-process depth 的 phenotype

### End-Stage Remodeled Phenotype / 晚期重构表型

**EN**
- end-stage HCM should be read as a remodeled phenotype
- fibrosis, chamber dilation, and vascular change matter here
- this is not simply “more hypertrophy”

**ZH**
- end-stage HCM 应被理解成一种重构后的表型
- fibrosis、chamber dilation 和 vascular change 在这里都重要
- 这不只是“更重的 hypertrophy”

### Comparative-Model Layer / 比较模型层

**EN**
- feline HCM can function as a spontaneous large-animal model of human HCM
- this model value belongs in the mechanism branch
- it should widen interpretation, not erase feline-specific structure

**ZH**
- 猫 HCM 可以作为 human HCM 的 spontaneous large-animal model
- 这种 model value 属于机制分支
- 它应扩宽解释层，而不应抹掉 feline-specific structure

## What We Can Defend / 当前可以稳妥主张的内容

**EN**
- HCM mechanism is best modeled as layered rather than single-cause
- phenotype, genetics, and remodeling all deserve visible top-level status
- genetics should deepen structure, not replace it
- remodeling should deepen phenotype, not replace it

**ZH**
- HCM 机制最适合被建模成分层结构，而不是 single-cause 故事
- phenotype、genetics 和 remodeling 都值得保留在顶层可见位置
- genetics 应加深结构解释，而不是替代结构
- remodeling 应加深 phenotype，而不是替代 phenotype

## What We Should Not Overstate / 当前不应说过头的内容

**EN**
- no single mutation explains all feline HCM
- no single remodeling story explains all phenotype heterogeneity
- mechanism depth should not outrank structure-first recognition

**ZH**
- 没有单一 mutation 能解释全部 feline HCM
- 没有单一 remodeling story 能解释全部表型异质性
- 机制深度不应压过 structure-first recognition
