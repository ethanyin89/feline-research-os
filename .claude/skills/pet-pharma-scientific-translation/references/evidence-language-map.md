# Evidence Language Map

本文件用于防止翻译时放大证据强度。

| English expression | 推荐中文 | 注意事项 |
|---|---|---|
| demonstrate | 显示；在证据充分时可译“证明” | 不默认译为“证实” |
| show | 显示 | 中性 |
| indicate | 表明；提示 | 根据证据强度选择 |
| suggest | 提示 | 不译为“证实” |
| support | 支持 | 常用于“支持某一假设” |
| may | 可能 | 必须保留不确定性 |
| might | 可能 | 通常比 may 更弱，不必机械区分但不得删除 |
| appears to | 似乎；初步显示 | 保留暂定性 |
| is associated with | 与……相关 | 不译为“导致” |
| correlates with | 与……相关 | 不等于因果 |
| predicts | 可预测；与后续结局相关 | 需看是否真正建立预测模型 |
| results in | 导致；产生 | 仅在因果语境明确时使用 |
| contributes to | 参与；促成；可能促进 | 不一定是唯一原因 |
| improves | 改善 | 必须明确改善的具体指标 |
| reduces | 降低 | 保留对象、幅度和时间点 |
| was not significant | 差异无统计学意义 | 不等于“无差异” |
| no evidence of | 未发现……证据 | 不等于“证实不存在” |
| proof of concept | 概念验证 | 不等于临床有效 |
| promising | 显示出一定前景；具有潜在价值 | 避免宣传化 |
| clinically actionable | 可用于临床决策 | 只有作者明确且证据充分时使用 |
| warrants further investigation | 值得进一步研究 | 不扩写为“应立即应用” |

## 因果与相关性

只有在以下条件较强时才考虑使用“导致、引起、使”：

- 研究设计具有明确干预或因果识别能力；
- 时间顺序明确；
- 比较组适当；
- 原文使用因果措辞且不过度外推。

观察性、横断面、病例对照和相关分析默认使用“相关、伴随、与……有关”。

## 证据成熟度

- exploratory：探索性
- hypothesis-generating：用于生成假设 / 假设生成性
- preliminary：初步
- proof-of-concept：概念验证
- externally validated：经外部验证
- clinically validated：经临床验证
- practice-changing：可能改变临床实践；仅在证据和语境确实支持时使用
