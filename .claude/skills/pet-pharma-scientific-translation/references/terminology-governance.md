# 术语治理规则

## 1. 术语优先级

选择译法时依次参考：

1. 中国现行法律法规、国家/行业标准及官方药品名称；
2. 权威专业指南、专家共识和教材；
3. 高质量中文论文中的稳定用法；
4. 国际指南和原始英文文献；
5. 在缺乏既定译名时采用语义透明、可回译的临时译法。

## 2. 一词一译

同一概念在同一文档中只保留一个主译法。若必须保留同义词，首次出现时说明，后续固定使用其中一个。

## 3. 不确定术语

无法确认权威中文名时：

- 保留英文；
- 给出暂定中文；
- 加译者注；
- 不伪装成已有官方译名。

## 4. 药物、基因和蛋白

- 药物通用名优先使用权威中文通用名；无权威译名时保留 INN/USAN 名称。
- 研发代码不得意译。
- 基因符号、蛋白符号按原文保留，并注意斜体/正体规范。
- 不能因为中文流畅而混淆基因、蛋白、通路和药物靶点。

## 5. 动物与研究语境

- “实验用猫”优先于“实验猫”。
- `client-owned cats` 不机械译作“客户拥有的猫”；科研正文可写“客户所有猫”，必要时补充“自然患病”。
- `purpose-bred cats` 译为“目的繁育猫”或“为研究目的繁育的猫”。
- `pet cats` 译为“家养猫”或“宠物猫”，视研究语境选择。

## 6. 临床研究术语

需固定区分：

- screening：筛查
- diagnosis：诊断
- staging：分期
- phenotyping：表型评估 / 表型分析
- risk stratification：风险分层
- prognosis：预后
- endpoint：终点
- outcome：结局
- efficacy：疗效（受控条件）
- effectiveness：实际效果 / 真实世界效果
- safety：安全性
- tolerability：耐受性

## 7. 每篇文档的术语表

长文翻译前复制 `templates/terminology-sheet.csv`，至少填写：

- source_term
- preferred_zh
- abbreviation
- domain
- first_mention
- avoid_zh
- source_or_rationale
