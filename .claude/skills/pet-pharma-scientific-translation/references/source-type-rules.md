# 不同科研文本类型的翻译规则

## Title

- 优先准确传达研究对象、变量、方法和目的。
- 允许调整词序，不允许添加评价。
- 保留研究设计关键词，如 randomized、prospective、retrospective、cross-sectional。

## Abstract

- 保留 Background / Objectives / Methods / Results / Conclusions 的功能边界。
- Results 中所有数字和比较关系必须保留。
- Conclusions 不得比 Results 更强；翻译时尤其避免进一步放大。

## Introduction

- 保留文献背景与研究缺口。
- 不把作者的研究动机写成已经得到的结论。
- 区分 established knowledge、controversy 和 knowledge gap。

## Materials and Methods

- 优先准确，不追求华丽。
- 保留试验设计、动物来源、纳入排除、随机、盲法、样本量、剂量、给药途径、时间点、设备、试剂和统计方法。
- 不省略品牌、型号和版本，除非用户明确要求精简。
- `n` 的含义必须明确：动物数、样本数、技术重复或生物学重复。

## Results

- 先数据，后解释。
- 保留所有比较对象、方向、效应量、离散程度和统计结果。
- 不在结果段加入讨论性判断。
- “差异无统计学意义”不写成“无差异”。

## Discussion

- 区分：本研究发现、与既往研究一致/不一致、机制解释、临床意义、局限、未来研究。
- 保留作者的保留措辞。
- 对外推、推测和假设使用较弱语言。

## Conclusion

- 不因中文追求有力而强化原文。
- 若原文结论超出研究设计，忠实翻译正文，同时可在独立译注中提示，但不要篡改原文。

## Figure and Table Legends

- 保留缩写解释、组别、样本、统计标记和显著性说明。
- 图表文字尽量简洁，但不能省略实验条件。

## Literature Review / Evidence Synthesis

- 标明观点归属。
- 区分原始研究、综述、指南和作者综合判断。
- 对冲突结果不得强行统一。
- 避免把“未检索到”写为“不存在”。
