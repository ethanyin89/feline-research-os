# 科研专业中文风格规范

## 1. 目标文体

目标不是逐词对应，而是在不改变证据含义的前提下，形成符合中文兽医学、生物医学和药理学论文习惯的表达。读者应感觉这是专业人员撰写的中文，而不是英文句法的镜像。

## 2. 句子组织

- 优先使用“研究对象—方法/条件—结果—解释边界”的顺序。
- 英文长句可拆分，但不能拆断因果、比较或限定关系。
- 避免连续三层以上“的”字定语，必要时改为分句。
- 英文被动句不必机械译为“被……”，可根据中文习惯转换为主动或无主句。
- 结果段优先陈述数据，不先写价值判断。
- 讨论段明确区分作者解释、研究结果和译者补充。

## 3. 科研常用表达

推荐：

- 本研究纳入……
- 研究采用……方法评估……
- 与对照组相比，……
- 结果显示……
- 结果提示……
- 该结果支持……这一假设
- 未观察到统计学显著差异
- 尚不能据此判断……
- 其临床意义仍需进一步验证
- 该研究的主要局限包括……

减少使用：

- 对……进行了研究
- 展现出了……的能力
- 有望成为……
- 具有巨大潜力
- 成功证明了……
- 令人鼓舞的是……

## 4. 数字、统计和单位

- 数字、方向、单位、范围、百分比、置信区间和 P 值必须与原文一致。
- `P < 0.05`、`95% CI`、`HR`、`OR`、`RR` 等保留标准写法。
- 不把“statistically significant”自动扩写为“具有临床意义”。
- 不把“not significant”译为“无差异”；应译为“差异无统计学意义”或“未观察到统计学显著差异”。
- 均值、标准差、中位数和四分位距不得混用。

## 5. 研究对象与模型

严格区分：

- client-owned animals：客户所有动物；在强调自然临床病例时可写“自然患病、客户所有的……”，但不得擅自省略 client-owned 属性。
- purpose-bred animals：目的繁育动物 / 为研究目的繁育的动物。
- laboratory animals：实验用动物。
- experimentally induced model：实验诱导模型。
- spontaneous disease model：自发性疾病模型。
- postmortem tissue：死后组织 / 尸检组织，按语境选择。
- cell model / iPSC-derived cells：细胞模型 / iPSC来源细胞，不能写成临床证据。

## 6. 证据层级表达

- 体外结果只说明细胞或分子层面的效应。
- 动物模型结果不能直接写为对临床患者有效。
- 客户所有动物的观察性研究仍不能自动支持因果关系。
- 随机对照试验可支持更强的干预结论，但仍需看样本、终点和随访。
- 横断面研究不能说明长期进展。
- 替代终点改善不能自动等同于生存或生活质量获益。

## 7. 首次出现和缩写

默认格式：

> 中文全称（English full term，缩写）

例如：

> 肥厚型心肌病（hypertrophic cardiomyopathy，HCM）

后文可直接使用 HCM。若中文领域已经高度约定俗成，可使用：

> 肥厚型心肌病（HCM）

不要在同一文档内交替使用不同中文全称。

## 8. 标题与小标题

中文标题应准确、克制、信息密度高。论文标题可在保持原义的前提下调整词序，不必逐词对应。标题中不擅自加入“新型、首次、突破、重要”等原文没有的评价。

## 9. 引用与图表

- 保留作者—年份或数字引用形式。
- 保留图、表、补充材料编号。
- 图注和表注不能省略限定条件、统计标记或缩写解释。
- 参考文献题名原则上不翻译，除非用户明确要求提供中文题名。
