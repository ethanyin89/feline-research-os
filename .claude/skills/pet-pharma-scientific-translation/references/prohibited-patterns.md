# 常见禁用或警示表达

以下表达并非绝对禁止，但出现时必须核查原文和证据强度。

## 1. 证据放大

- 证实了
- 明确证明
- 必然导致
- 完全解决
- 可直接用于临床
- 已成为标准治疗
- 显著改善预后

## 2. 宣传化

- 突破性
- 颠覆性
- 革命性
- 里程碑式
- 前所未有
- 极具潜力
- 令人振奋

## 3. 翻译腔

- 展示出了……的能力
- 对……进行了……的研究
- 在……方面表现出
- 其结果是……的
- 这表明了……这一事实
- 被观察到具有……

## 4. 科学概念混淆

- 把 statistically significant 写成“临床显著”
- 把 associated with 写成“导致”
- 把 no evidence 写成“不存在”
- 把 exploratory 写成“已验证”
- 把 animal model 写成“临床研究”
- 把 proof-of-concept 写成“临床有效”
- 把 safety signal 写成“已证实风险”

## 5. 动物研究用语

- “实验猫”作为默认规范用语；优先“实验用猫”
- “客户拥有的猫”；优先“客户所有猫”
- “特制繁育猫”；优先“目的繁育猫”
