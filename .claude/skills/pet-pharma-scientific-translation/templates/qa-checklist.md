# 双语科研翻译 QA Checklist

## A. 严重错误

- [ ] 是否遗漏整句、整段、表格行、图注或脚注？
- [ ] 是否添加了原文没有的结果、机制或临床建议？
- [ ] 研究对象、物种、品种、组别是否正确？
- [ ] 治疗组与对照组是否颠倒？
- [ ] 增加/降低、阳性/阴性、显著/不显著方向是否正确？
- [ ] 给药剂量、频次、途径和时间点是否正确？

## B. 数字与统计

- [ ] 样本量是否一致？
- [ ] 百分比、均值、中位数、范围、SD、SE、IQR 是否一致？
- [ ] P 值、CI、HR、OR、RR 是否一致？
- [ ] 单位是否一致，是否发生数量级错误？
- [ ] “无统计学意义”是否被误写成“无差异”？

## C. 证据强度

- [ ] association 是否被误译为 causation？
- [ ] may / might / suggest / appear 是否被删除或强化？
- [ ] 动物模型是否被误写为临床有效？
- [ ] 替代终点是否被误写为结局获益？
- [ ] 探索性标志物是否被写成可直接应用？
- [ ] 未检出是否被写成不存在？

## D. 术语

- [ ] 同一概念是否保持同一译法？
- [ ] 首次出现是否给出全称和缩写？
- [ ] 药物、基因、蛋白和产品名称是否准确？
- [ ] “实验用猫”“客户所有猫”“目的繁育猫”等是否按语境正确使用？

## E. 文体与结构

- [ ] 标题层级、段落、表格、图注和引用是否保留？
- [ ] 是否存在明显英文语序？
- [ ] 是否存在过长定语链或连续被动句？
- [ ] 是否出现无依据的宣传性词语？
- [ ] 是否把解释混入了忠实译文？

## F. 最终交付

- [ ] 所有待确认术语是否显式标记？
- [ ] 译者注是否与正文分离？
- [ ] 是否说明版本、日期和修订范围？
- [ ] 是否完成独立的第二遍核对？
