# FCV Triggers Mitochondrial Apoptosis Pathway

## Evidence-Depth Caveat

Abstract-weighted extraction. Full text needed for complete kinetics data.

## One-Line Summary

FCV activates mitochondrial apoptosis: Bax translocation (4-6h) → cytochrome c release (6-8h) → caspase-9/3 activation.

## Why It Matters For Feline FCV

Explains cellular mechanism of FCV cytopathology; intrinsic apoptosis pathway.

## Key Findings

### quoted_fact

- 4-6h: Bax translocation, mitochondrial membrane potential loss
- 6-8h: Phosphatidylserine flip, cytochrome c release
- Sequential: caspase-9 → caspase-3 activation

### source_supported_conclusion

- FCV uses intrinsic (mitochondrial) apoptosis pathway
- Clear temporal sequence of apoptotic events
