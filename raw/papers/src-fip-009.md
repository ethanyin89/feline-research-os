## Why This Study Matters

This paper is one of the central mechanism anchors in the FIP set because it frames FIP virus emergence as mutation from endemic feline enteric coronavirus background. That immediately changes how the disease should be organized. FIP is not best modeled as simple exposure to an unrelated agent. It is better modeled as disease emergence within a broader coronavirus ecology.

The main value is mechanistic architecture. The paper helps define an upstream branch: endemic FECV background can give rise to pathogenic FIP virus through mutation and systemic emergence. That branch should appear before later questions about spike changes, 3c changes, CSF testing, or diagnostic mutation panels. If diagnostic papers are read without this anchor, the module can collapse into assay logic and lose the disease-emergence story.

The main boundary is just as important. A mutation-origin paper does not automatically prove that a given mutation test is clinically definitive. Mechanism and diagnosis are related but not identical. This source should therefore support the mechanism overview and synthesis index, while later sources such as `src-fip-010`, `src-fip-018`, and `src-fip-022` control diagnostic utility and mutation-limit language.

The safe compiled rule is: mutation-origin belongs to mechanism first; mutation-based diagnosis comes later and must remain bounded by test-performance and compartment-specific evidence.

For navigation, this card should be read before the 3c, spike-spread, and spike-mutation diagnostic papers. It defines the origin branch; those later sources define refinements, limits, and clinical test boundaries. Keeping that order prevents the mechanism branch from being reduced to a list of candidate mutations.
