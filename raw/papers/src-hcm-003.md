# One-line Summary

Comparative framing source that positions feline HCM as a human-HCM model.

## Why It Matters For HCM

- introduces a cross-species model branch early
- may shape how strongly the vault uses feline HCM for translational interpretation

## Key Findings

- explicit human-model framing
- abstract reports 10-15% prevalence in the pet cat population
- abstract frames feline HCM as genotypically and phenotypically similar to human HCM
- Maine Coon and Ragdoll cats are highlighted as especially valuable models due to myosin binding protein-C mutations and higher disease incidence
- the source explicitly connects feline endpoints with translational model value
- strongest use is comparative mechanism / model bridge, not feline clinical hierarchy transfer

## Why This Study Matters

This paper matters because it gives the HCM module a cleaner model branch than generic comparative language alone can provide. The title is already unusually explicit: feline HCM is framed as a spontaneous large-animal model of human HCM.

That matters structurally. It gives the vault a way to talk about why feline HCM is interesting beyond feline clinical management. But it also creates the main risk: once model value is stated strongly, it becomes easy to smuggle in unearned human-to-cat transfer.

So the safest use of this source is dual. It should strengthen the comparative branch and the translational imagination of the module, while simultaneously making the need for feline-first hierarchy more explicit.

## Model-Bridge Boundary

The strongest reusable write-back from this source is:

- feline HCM has a real spontaneous large-animal model branch
- comparative-model value belongs in mechanism and synthesis architecture
- model usefulness should not collapse into direct clinical hierarchy transfer
- strong human relevance requires stronger translational caution, not less

That is the right level for this paper: important bridge logic with bounded authority.

## Model Strengths

This source gives the HCM module a unusually direct model argument:

- feline HCM is common enough to matter as naturally occurring disease
- the disease has defined endpoints
- the cat model carries both genotype and phenotype similarity to human HCM
- Maine Coon and Ragdoll mutation contexts provide particularly strong model subbranches
- the model can bridge smaller experimental systems and human clinical trial imagination

These points make the source stronger than a generic comparative note.

## Model Overclaim Boundary

The same strengths create a specific risk.

If the source is reused badly, it could make the module sound as though:

`human HCM logic automatically transfers into feline clinical care`

That is not the safe read.

The safe read is:

`feline HCM has real translational model value, but feline clinical hierarchy still requires feline evidence`

This distinction should govern mechanism, translation, and treatment pages.

## Limits / Caveats

- source-page checked: the 10-15% prevalence, model-similarity, Maine Coon / Ragdoll, and endpoint claims are abstract-backed; which specific phenotypic and genetic features drive the model claim in detail requires full-text review
- model framing does not transfer human clinical hierarchy to feline practice without explicit feline evidence
- the paper's 2017 date means genetics-heavy human HCM literature (post-sarcomere-mutation era) is likely incorporated, but exactly how is abstract-inaccessible
- this card supports comparative-model branch existence and the prevalence anchor more strongly than any specific similarity ranking across phenotype, genetics, or intervention depth
