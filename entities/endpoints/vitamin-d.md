# Vitamin D

## Why It Matters

Vitamin D matters because it is one of the clearest current systemic burden markers spanning feline IBD and small-cell alimentary lymphoma.

## Current Notes

- In the current vault, vitamin D belongs to a shared support-marker layer rather than to a discriminatory boundary layer.
- It is currently more useful for burden, malabsorption, or complication-context interpretation than for disease-class separation.
- It should sit below exclusion-first workup and below frontier metabolomic separation claims.
