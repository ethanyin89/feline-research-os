# Potassium

## Why It Matters

Potassium matters because hypokalaemia is a clinically important CKD complication branch, and potassium correction is part of supportive-care logic in feline CKD.

## Current Notes

- Potassium is not a first-wave CKD diagnostic anchor.
- Its main value is treatment-context and complication-control logic.
- The current vault supports potassium correction as clinically meaningful when hypokalaemia is present.
- Potassium should be treated as a supportive-care endpoint rather than a lead disease-modification signal.
