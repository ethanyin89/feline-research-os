# Mutation Testing

## Why It Matters

Mutation testing matters because it is one of the main supportive diagnostic channels in the current FIP workup, but it is also one of the highest-risk overclaim branches.

## Current Notes

- Mutation testing is relevant in the current vault as bounded diagnostic strengthening.
- It should be used below certainty language.
- The current module explicitly separates mutation-origin, mutation-detection utility, and mutation-diagnostic limitation.
