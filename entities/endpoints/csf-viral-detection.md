# CSF Viral Detection

## Why It Matters

CSF viral detection matters because it is the clearest current marker branch that forces neurologic FIP workup to be treated as a specialized extension rather than ordinary FIP suspicion.

## Current Notes

- In the current vault, CSF viral detection belongs to the neurologic-extension branch.
- It should not be generalized into the default FIP workup.
- Its strongest value is changing the workup shape when CNS complexity is present.
