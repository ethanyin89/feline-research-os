# Small-Cell Alimentary Lymphoma

## Why It Matters

Small-cell alimentary lymphoma matters because it is the main disease-boundary pressure that prevents feline IBD from being compiled as a simple inflammatory syndrome.

## Current Notes

- In the current vault, small-cell lymphoma is not a side differential. It is built into the core workup architecture.
- Muscularis thickening, biopsy-site selection, and several tissue-marker papers all exist primarily because this branch is so close to IBD.
- The current compiled state still treats the boundary as multimodal rather than marker-led.
