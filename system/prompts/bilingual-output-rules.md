# Bilingual Output Rules

Use these rules when deriving English outputs from Chinese-first internal material.

## Core Rules

1. Keep the evidence layer labels unchanged in structure:
   - `quoted_fact`
   - `source_supported_conclusion`
   - `llm_inference`

2. Translate for scientific clarity, not for literary style.

3. Keep technical terms in standard English form:
   - urine specific gravity
   - UPCR
   - systolic blood pressure
   - renal fibrosis

4. Do not strengthen claims during translation.

5. If the Chinese source uses a softer conclusion, keep it soft in English.

6. If a regulatory point is route-level only, do not rewrite it as indication-specific.

## Minimum Footer Practice

Every bilingual derivative should make clear:

- it is derived from an existing source output
- source ids are unchanged unless expanded
- translation does not imply stronger evidence than the original
