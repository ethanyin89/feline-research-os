## 什么是猫慢性肾病（CKD）？

**猫慢性肾病是指猫的肾脏长期受损，逐渐失去正常功能的疾病。**

肾脏的工作是过滤血液、排出废物、调节水分和矿物质。当肾脏受损后，这些功能会慢慢下降，废物在体内堆积，引起各种健康问题。

[source_supported_conclusion: src-ckd-004]

## 有多常见？

CKD是老年猫最常见的疾病之一：
- **10岁以上**的猫中，约有**30-40%**患有某种程度的肾病
- 随着猫的寿命延长，CKD发病率也在增加
- 这是老年猫最常见的死亡原因之一

[source_supported_conclusion: src-ckd-001]

## 有什么症状？

| 常见症状 |
|----------|
| 喝水增多 |
| 排尿增多 |
| 体重下降 |
| 食欲减退 |
| 呕吐 |
| 精神变差 |
| 毛发粗糙 |

**注意：** 这些症状通常是渐进的，早期可能不明显。如果发现这些症状，请咨询兽医。

[source_supported_conclusion: src-ckd-004]

## 如何诊断？

CKD诊断需要综合多项检查：
| 检查项目 | 作用 |
|----------|------|
| 血液检查 | 评估肾功能（肌酐、尿素氮） |
| 尿液检查 | 评估肾脏浓缩能力 |
| 血压测量 | 高血压与CKD相关 |
| 超声波 | 观察肾脏结构 |

[source_supported_conclusion: src-ckd-004, src-ckd-010]

## 可以治疗吗？

**CKD目前无法治愈，但可以管理。**

目标是减缓病情进展、控制症状、提高生活质量。

| 管理措施 | 作用 |
|----------|------|
| 肾脏处方粮 | 最重要的基础管理 |
| 磷结合剂 | 控制血磷 |
| 抗高血压药 | 控制血压 |
| 补液治疗 | 维持水分平衡 |

**关键：** 肾脏处方粮是最有证据支持的管理措施。

[source_supported_conclusion: src-ckd-003, src-ckd-004]

## 早期发现很重要

- **7岁以上**的猫应该定期体检
- 包括血液和尿液检查
- 早期发现可以更早开始管理
