`the IBD module is stronger when its adjacent diseases are modeled as edge branches, not absorbed into the core idiopathic spine`

## What The Module Can Already Say

- extension branches help define where the IBD module ends
- module safety improves when neighboring inflammatory, remodeling, and hepatobiliary diseases are kept visible but separate
- core IBD becomes easier to defend when adjacent diseases are modeled as edge branches rather than hidden inside the main synthesis

## What The Module Should Not Yet Say

- the extension branch is not the same thing as a finished triaditis architecture
- extension diseases should not rewrite the core IBD-versus-lymphoma boundary
- practice framing should not be mistaken for high-strength biologic or treatment evidence
