# CKD Proteinuria-Oriented Renoprotective Route Memo

Date: 2026-04-09  
Scope: proteinuria-oriented renoprotective adjunct viewed through current route-level regulatory framework

This memo is the third product-specific route memo in the CKD vault.

It uses a borderline drug-like archetype:

- proteinuria-oriented RAAS / renoprotective adjunct logic in feline CKD

This page is an English route memo.
It is a route-fit working document, not a final jurisdiction recommendation and not a bilingual summary.

## Product Definition Used Here

### Product archetype

- proteinuria-oriented renoprotective adjunct
- veterinary medicine-style adjunct for cats with CKD and clinically relevant proteinuria

### Best-fit claim style in the current vault

- proteinuria reduction
- management of a higher-risk CKD subgroup
- adjunctive risk-context improvement

### Claim style to avoid for now

- broad disease-modification claims
- broad renal-survival claims
- wording that implies all proteinuric cats form one homogeneous subgroup

## Why This Archetype Is The Right Third Route Memo

- it is still drug-like enough to fit the current medicine-centered regulatory corpus
- it is more overclaim-sensitive than hemodynamic management
- it forces the vault to test where subgroup clarity starts to blur
- it also exposes where treatment logic and pathology-subset logic begin to overlap

## What This Memo Changes

This memo shows a middle pattern between the first two route memos:

`the archetype still fits the current regulatory corpus, but the claim discipline burden rises sharply once proteinuria is asked to carry broad renoprotective meaning`

That makes it a useful stress test for route-fit limits.

## Best Current Internal Read

### What we can say now

- this archetype is medicine-like enough to work inside the current route-level corpus
- FDA remains the cleanest current branch because route-fork and study-design thinking are already available there
- EMA and VMD remain usable for structured comparison
- subgroup logic helps, but does not by itself solve evidentiary weakness

### What we should not say now

- do not claim this archetype already supports a jurisdiction recommendation
- do not claim proteinuria reduction automatically upgrades into proven broad renoprotection
- do not let glomerular-subset logic disappear inside one generic CKD adjunct label
- do not read route usability as proof that the treatment claim is already clean

## Decision Logic This Memo Supports

1. if the asset is framed narrowly around proteinuria, route discussion remains possible
2. if the claim expands into broad renoprotection, route-fit clarity falls quickly
3. subgroup language helps only when claim discipline stays tight
4. route comparison should follow claim tightening, not substitute for it

## Most Important New Insight

The main bottleneck for this archetype is not only jurisdiction.

It is that:

`route-fit can stay usable while claim-fit is already drifting out of bounds`

That is why this archetype is high-value.
It exposes the difference between a route-capable product story and a defensible evidence story.
