# HCM Genotype-Severity Memo

- Date: `2026-04-10`
- Scope: `src-hcm-004`, `src-hcm-012`, `src-hcm-001`, `src-hcm-007`

This memo exists to stop the HCM genetics branch from collapsing into one flat sentence like:

`there are HCM mutations`

That sentence is directionally true, but it is structurally too weak for this vault.

## Core Takeaway

`the current HCM genetics branch is strong enough to compile as dosage-sensitive and age-penetrant genotype pressure, but not strong enough to become a one-mutation explanation of all feline HCM`

## Current Genotype-Severity Layers

### Layer 1: Broad Genetic Branch

This layer is mainly carried by:

- `src-hcm-004`
- `src-hcm-007`

Current meaning:

- genetics is a first-class branch in the HCM module
- HCM should not be flattened into phenotype alone
- broad cardiomyopathy framing helps keep genotype inside a wider myocardial-disease map

### Layer 2: Dosage-Sensitive Severity Pressure

This layer is mainly carried by:

- `src-hcm-012`

Current meaning:

- heterozygous and homozygous mutation states are not clinically equivalent
- mutation dosage should remain visible in compiled HCM language
- genotype pressure should connect directly to severity and cardiac-death risk, not just susceptibility

### Layer 3: Age-Related Penetrance

This layer is mainly carried by:

- `src-hcm-012`
- `src-hcm-001`

Current meaning:

- mutation-positive or genetically pressured disease should not be modeled as temporally flat
- age-aware penetrance belongs in recognition logic
- genotype risk and current phenotype are related but not interchangeable

## What The Module Can Already Say

- genetics belongs near the top layer of HCM explanation
- genotype pressure is better modeled as `stratified`, not binary
- mutation dosage and age-related penetrance already make the recognition branch more nuanced
- genetics should deepen phenotype interpretation, not replace structural recognition

## What The Module Should Not Yet Say

- there is no safe one-mutation explanation of all feline HCM
- there is no safe universal genotype-to-prognosis rule across all breeds and variants
- mutation status should not be promoted into standalone diagnostic certainty
