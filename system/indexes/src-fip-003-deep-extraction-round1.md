## Phase 1: Theme Reconstruction

## Theme: FIP should be modeled as a systems problem, not a single-test problem

The strongest reusable contribution of this review is not one isolated result. It is that FIP remains a disease where diagnosis depends on integrating imperfect signals. That means the module should not be built around one favored assay, one mutation, or one symptom.

### Hard Information

- abstract states no specific symptoms
- abstract states laboratory changes are non-specific
- abstract supports supportive diagnosis framing

## Theme: Coronavirus exposure and FIP disease emergence are different layers

This review supports a structural distinction the vault needs early: exposure to feline coronavirus background does not equal FIP. The disease map should separate endemic enteric coronavirus context from systemic and pathogenic FIP emergence.

### Hard Information

- abstract distinguishes serotypes and pathotypes
- feline enteric coronavirus and FIP virus are not treated as one flat category

## Theme: Broad review framing should come before mutation or treatment specialization

Because this source is broad and recent, it is the right first anchor for vocabulary, branch layout, and safe baseline claims. It should stabilize the module before mutation papers and antiviral papers start to pull the map in narrower directions.

### Hard Information

- 2024 publication date
- whole-disease review framing

## Theme: Diagnostic ambiguity and treatment transformation must coexist in the same module

This review is useful because it likely bridges old truths and new realities: FIP remains diagnostically difficult, but the disease can no longer be modeled only as an untreatable endpoint. This is one reason FIP is a strong next disease module after CKD.

### Hard Information

- broad review timing is modern
- abstract still foregrounds diagnostic difficulty
