# IBD Fibrosis Memo

## Core Takeaway

`fibrosis should be modeled as a structural chronicity branch, not as background histology`

## Strongest Current Anchor

- [src-ibd-022 deep extraction round 1](src-ibd-022-deep-extraction-round1.md)

## What The Module Can Already Say

- fibrosis is common enough in feline chronic inflammatory enteropathy to deserve explicit modeling
- fibrosis tracks burden-related features such as lower body weight and lower albumin
- chronic-enteropathy architecture becomes stronger when structural remodeling is separated from activity and support-marker layers
- extension remodeling phenotypes such as eosinophilic sclerosing fibroplasia can widen the remodeling map without being merged into the core fibrosis branch

## Current Boundaries

- fibrosis is not the same thing as inflammatory activity
- fibrosis is not the same thing as a ready-made antifibrotic treatment route
- fibrosis does not by itself resolve the lymphoma boundary
- specialized fibroplasia phenotypes should remain extension branches rather than being collapsed into idiopathic IBD
