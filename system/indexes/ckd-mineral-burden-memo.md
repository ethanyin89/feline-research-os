# CKD Mineral-Burden Memo

- Date: `2026-04-09`
- Scope: `src-ckd-006`, `src-ckd-015`, `src-ckd-003`

This memo exists to move the mineral branch one level above phosphorus alone.

In the current vault, phosphorus is still the strongest mineral endpoint.
But feline CKD mineral burden is broader than phosphorus, and the branch now clearly includes:

- phosphate retention
- PTH-mediated compensation
- calcium dysregulation
- hypercalcaemia risk
- FGF23-led early mineral-regulator change

## Core Position

1. Mineral burden in feline CKD should be modeled as a branch, not as a single marker problem.
2. Phosphorus remains the strongest current operational endpoint inside that branch.
3. PTH, calcium dysregulation, and FGF23 widen interpretation even when they are not first-wave routine endpoints.
4. The current vault is stronger at explaining mineral-burden logic than at turning the whole branch into a clean intervention package.

## What This Memo Separates

### Operational endpoint logic

This is where phosphorus sits.

- strongest routine mineral endpoint
- clinically useful for monitoring and progression framing
- treatment-relevant and survival-linked in current sources

### Interpretive context logic

This is where PTH currently sits.

- helps explain earlier phosphate-retention biology
- improves interpretation when serum phosphorus alone looks reassuring
- still below first-wave default endpoint status

### Expanded mineral-regulation logic

This is where calcium dysregulation and FGF23 enter.

- calcium / hypercalcaemia branch adds tradeoff awareness
- FGF23 widens the branch earlier in disease evolution
- neither is yet ready to displace phosphorus as the main operational mineral endpoint

## What The Current Sources Actually Support

### Stronger Support

- phosphorus should stay in the top-tier endpoint set
- mineral burden should not be reduced to a treatment-only or chemistry-only footnote
- CKD-MBD belongs in the interpretive frame around phosphorus

### Bounded Support

- FGF23 is important, but not yet a first-wave routine endpoint in this vault
- calcium / hypercalcaemia logic is clinically meaningful, but still not operationally clean enough for top-tier endpoint status
- the branch is better for interpretation than for direct product strategy at this stage

## What This Branch Is Not Allowed To Become

- It is not allowed to become a phosphorus-only story.
- It is not allowed to promote FGF23 into a first-wave routine endpoint before the current evidence layer supports that move.
- It is not allowed to flatten calcium/hypercalcaemia risk into a minor side note.
- It is not allowed to assume mineral-burden explanation automatically produces a clean intervention strategy.

## What The Vault Can Now Say More Clearly

1. Phosphorus is still the operational center of the mineral branch.
2. PTH helps explain hidden mineral burden before phosphorus alone fully changes.
3. Calcium dysregulation and hypercalcaemia risk mean mineral management is a balance problem.
4. FGF23 expands the branch earlier into disease evolution, but remains below routine first-wave endpoint status.
5. CKD-MBD should now be treated as a real interpretive branch in the vault, not as a niche side topic.
