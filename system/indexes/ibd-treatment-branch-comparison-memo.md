# IBD Treatment-Branch Comparison Memo

- Date: `2026-04-10`
- Scope: `src-ibd-014`, `src-ibd-021`, `src-ibd-005`

This memo exists because the current IBD translation layer is now repeatedly being forced to answer a narrower question:

`how should the direct practical anchor, broad overview context, and exploratory translational branch be compared without flattening them into one treatment story?`

## Core Takeaway

`the IBD treatment layer is now clearer as a branch-comparison problem than as a flat treatment-ranking problem`

## Current Branches

### Branch 1: Direct Practical Anchor

Main anchor:

- `src-ibd-014`

Current meaning:

- this remains the cleanest current practical branch
- `diet-first chronic-enteropathy management` is the safest ordinary compiled frame
- this branch still overlaps with food-responsive disease and should not be upgraded into idiopathic-IBD-specific certainty

### Branch 2: Broad Therapeutic-Overview Context

Main anchor:

- `src-ibd-021`

Current meaning:

- this branch widens the treatment architecture
- it helps place diet, immunomodulation, microbiota-directed ideas, and regenerative ideas inside one larger map
- it does not lead feline-primary efficacy hierarchy

### Branch 3: Exploratory Translational Branch

Main anchor:

- `src-ibd-005`

Current meaning:

- this branch is real enough to keep in the module
- it belongs in translational context rather than routine guidance
- it sharpens future possibility more than current management authority

## What This Comparison Clarifies

- `direct practical anchor` remains the routine leader
- `broad overview context` helps orient the map, but does not outrank the practical anchor
- `exploratory translational branch` belongs in the module because it clarifies future direction, not because it leads current care

## What The Module Can Already Say

- IBD treatment should be read as three non-equivalent branches, not one ladder
- the ordinary compiled default should still lead with `diet-first chronic-enteropathy management`
- broad overview material matters because it prevents the treatment map from shrinking to one diet line
- exploratory translational work belongs in the module because it widens future branch structure without overrunning current evidence

## What The Module Should Not Yet Say

- there is still no safe final treatment ranking
- broad overview material should not be promoted into feline-primary efficacy authority
- exploratory translational branches should not be promoted into routine-ready management
- current treatment language should not outrun the exclusion-first diagnostic spine
