# FIP Clinicopathology Memo

- Date: `2026-04-09`
- Scope: `src-fip-006`, `src-fip-015`

This memo exists to stop the FIP module from talking about clinicopathology as though it were only a vague supporting idea.

## Core Position

1. Clinicopathology is the leading operational recognition layer in the current FIP vault.
2. Clinicopathology should lead suspicion-pattern reading before mutation or CSF-based strengthening enters.
3. Disease-form-aware recognition matters; FIP should not be read as one flat presentation.
4. Clinicopathology is highly operational, but it is still not definitive diagnosis.

## Current Clinicopathology Layers

### Layer 1: Practical Pattern Recognition

This layer is mainly carried by:

- `src-fip-006`

What it means:

- real-world case patterns matter
- wet versus dry suspicion likely needs separate attention
- smaller descriptive series still stabilize recognition architecture

### Layer 2: Structured Staging And Disease-Form Recognition

This layer is mainly carried by:

- `src-fip-015`

What it means:

- the branch is not only descriptive, but increasingly structured
- disease staging and form-aware recognition can be modeled explicitly
- recognition can become more patterned without becoming certainty

## Current Working Hierarchy

| Clinicopathology Branch | Current Position In Vault | Notes |
|---|---|---|
| Staged clinicopathology and disease-form recognition | strongest current recognition anchor | larger and more structured |
| Practical clinicopathologic case-pattern recognition | confirming operational anchor | older but still highly reusable |
| Molecular strengthening | downstream from clinicopathology | should not lead suspicion |
| Neurologic extension | separate branch shift | should not replace ordinary suspicion-pattern logic |

## What The Current Sources Actually Support

### Stronger Support

- clinicopathology should sit near the center of FIP recognition
- disease-form-aware recognition is more useful than one generic pattern sentence
- smaller older case series and larger later staging series can cooperate rather than compete

### Bounded Support

- clinicopathology remains suspicion-building, not certainty
- older cohorts do not directly mirror modern treated-era presentation
- staged disease-form recognition is still not a one-profile diagnostic shortcut

## What This Branch Is Not Allowed To Become

- It is not allowed to become one generic phrase like `clinicopathology matters`.
- It is not allowed to be demoted below mutation-related testing as though recognition starts molecularly.
- It is not allowed to be promoted into definitive diagnosis.
- It is not allowed to flatten effusive and non-effusive recognition into one presentation.

## What The Vault Can Now Say More Clearly

1. Clinicopathology is the main front door of suspicion-pattern recognition in FIP.
2. `src-fip-006` and `src-fip-015` form a two-step clinicopathology spine: practical pattern recognition plus structured staging.
3. Recognition architecture should start with what the disease looks like before asking which bounded test might strengthen it.
4. The right sentence is `clinicopathology leads suspicion`, not `clinicopathology proves diagnosis`.
