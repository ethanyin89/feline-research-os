# CKD Phosphorus-Control Evidence Memo

- Date: `2026-04-09`
- Scope: `src-ckd-006`, `src-ckd-015`, `src-ckd-003`, `src-ckd-007`

This memo exists to separate three things that are easy to flatten into one story:

1. renal-diet phosphorus logic
2. binder / adjunct phosphorus logic
3. broader CKD-MBD burden logic

In the current vault, phosphorus is one of the strongest bridge variables in feline CKD.
But that does not mean every phosphorus-related intervention or mineral-regulation branch has the same evidentiary weight.

## Core Position

1. Phosphorus belongs near the center of feline CKD treatment reasoning because it connects progression, secondary hyperparathyroidism, and survival-oriented interpretation.
2. Phosphorus-control diet logic is currently stronger than binder / adjunct logic in the feline evidence layer.
3. CKD-MBD logic is broader than phosphorus alone, so phosphorus control should not be framed as a one-variable optimization problem.
4. The vault can support strong endpoint importance here more easily than broad product-class certainty.

## Layer 1: Renal-Diet Phosphorus Logic

This is the cleanest phosphorus-control branch in the current vault.

What current sources support:

- phosphate retention is a major contributor to CKD progression
- dietary phosphorus restriction has meaningful support in feline stage 2-3 disease
- phosphorus burden is tied to secondary hyperparathyroidism logic

What the vault can say safely:

- phosphorus-focused diet logic belongs in Tier 1 treatment reasoning

What the vault should not say:

- do not treat renal-diet logic as equivalent to every phosphorus-control product branch

## Layer 2: Binder / Adjunct Phosphorus Logic

This branch is real, but weaker.

What current sources support:

- intestinal phosphate binders are widely relevant in practice
- binder use makes sense when diet alone is insufficient
- veterinary safety and efficacy literature is thinner than for diet
- no phosphorus binders were identified as licensed medications for dogs or cats in the reviewed source set

What the vault can say safely:

- binder / adjunct logic belongs in the treatment map
- it is more conditional and less strongly supported than diet logic

What the vault should not say:

- do not flatten binder evidence up to renal-diet strength
- do not assume category clarity just because phosphorus is a strong endpoint

## Layer 3: CKD-MBD Burden Logic

This layer is broader than a treatment class.

`src-ckd-015` matters because it widens the mineral branch:

- calcium dysregulation matters
- hypercalcaemia risk matters
- FGF23 belongs in the branch
- phosphorus-only framing is too narrow

What that changes:

- phosphorus remains central
- but phosphorus control is not the entire mineral-management story
- intervention reasoning now has to tolerate tradeoffs and cross-mineral interactions

## What The Current Sources Actually Support

### Stronger Support

- phosphorus should remain in the top-tier endpoint set
- diet-based phosphorus control is one of the strongest current treatment branches
- phosphorus belongs in any serious CKD progression or mineral-burden conversation

### Bounded Support

- adjunct binder logic is coherent but thinner
- mineral-regulation biology is broader than routine first-wave endpoint practice
- some mineral-management actions can create tradeoffs rather than pure one-direction benefit

## What This Branch Is Not Allowed To Become

- It is not allowed to become “phosphorus explains all mineral burden.”
- It is not allowed to become “diet and adjunct product evidence are interchangeable.”
- It is not allowed to become “phosphorus lowering equals proven disease modification.”
- It is not allowed to ignore hypercalcaemia and wider CKD-MBD tradeoffs.

## What The Vault Can Now Say More Clearly

1. Phosphorus is one of the strongest CKD bridge variables in the whole vault.
2. Renal-diet phosphorus logic is stronger than binder / adjunct logic.
3. CKD-MBD should be treated as a wider mineral-regulation branch, not only a phosphorus branch.
4. The treatment layer is cleaner than the product-category layer for phosphorus-control adjuncts.
