# Feline Cardiomyopathy Supermodule

- Date: `2026-04-10`
- Current depth: `HCM-rich, non-HCM-thin`

This page is the smallest usable top layer above the current HCM module.

It does not pretend that the vault already has a full feline-cardiomyopathy taxonomy.
It does something narrower:

- keeps HCM inside a broader cardiomyopathy map
- clarifies what is already thick versus still thin
- gives a stable landing page if later non-HCM cardiomyopathy branches are added

## Core Position

1. HCM is currently the deepest feline cardiomyopathy branch in the vault.
2. HCM should not silently absorb the rest of feline cardiomyopathy.
3. The correct current architecture is:
   - broad cardiomyopathy frame
   - HCM core spine
   - non-HCM extension branch
4. The supermodule is real enough for orientation, but not yet for a full taxonomy handbook.
