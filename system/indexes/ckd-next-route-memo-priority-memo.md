## Current Read Across The Four Route Memos

### Renal Diet

- strongest treatment archetype overall
- weaker route-fit than its treatment strength suggests
- useful mainly because it proved that best treatment logic does not equal best route logic

### Hemodynamic Management

- cleanest current route-fit archetype
- already proves that drug-like products match the current official-source pack better
- no longer the best `borderline` test because it is already too clean relative to the rest

### Proteinuria-Oriented Renoprotective Adjunct

- still drug-like enough to fit the current route corpus
- exposes where route-fit remains usable while claim-fit starts to drift
- best current stress test for the boundary between route-capable product logic and overclaim-sensitive treatment logic

### Phosphorus-Control Adjunct

- endpoint-fit is strong
- category-fit is still blurrier than the endpoint itself
- the current route memo already says the next useful move is not another route memo, but a thicker evidence layer

## Current Priority Order

### 1. Proteinuria-Oriented Renoprotective Adjunct

Why:

- it is genuinely borderline
- it still tests route usefulness without being as clean as hemodynamic management
- it exposes the most instructive failure mode now in the CKD regulatory layer:

`route-fit can stay usable while claim-fit is already drifting out of bounds`

### 2. Hemodynamic-Adjacent Borderline Refinement Only

Why:

- the hemodynamic branch is already doing its job as the cleanest route-fit proof
- more work here is more likely to repeat the same lesson than to sharpen a new boundary

### 3. Phosphorus-Control Adjunct: Pause At Route Level

Why:

- the current memo already reaches the route-layer carry limit
- the next gain should come from evidence architecture, not another route comparison
- phosphorus now needs thicker separation between renal-diet phosphorus logic, binder logic, and CKD-MBD burden logic

### 4. Renal Diet: No Immediate New Route Memo

Why:

- this branch already proved the mismatch between treatment strength and route cleanliness
- another route memo here would be lower-yield than proteinuria stress testing

## What The Module Can Already Say

- the next CKD route memo should only happen if it sharpens a genuine borderline archetype
- proteinuria renoprotection is currently the highest-yield borderline candidate
- phosphorus-control should now densify its evidence architecture before more route work
- hemodynamic management has already done enough to prove the clean-route side of the comparison

## What The Module Should Not Yet Say
