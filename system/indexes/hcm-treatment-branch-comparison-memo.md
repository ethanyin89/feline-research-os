# HCM Treatment-Branch Comparison Memo

- Date: `2026-04-10`
- Scope: `src-hcm-008`, `src-hcm-011`, `src-hcm-014`, `src-hcm-015`, `src-hcm-020`

This memo exists because the current HCM translation layer is no longer missing only a treatment page.

It is now repeatedly being forced to answer a narrower question:

`how should bounded management, targeted frontier, older conventional therapy, and evidence skepticism be compared without flattening them into one ranking?`

## Core Takeaway

`the HCM treatment branch is now clearer as a branch comparison problem than as a flat intervention hierarchy problem`

## Current Branches

### Branch 1: Bounded Management Anchor

Main anchor:

- `src-hcm-008`

Current meaning:

- this remains the safest routine-facing treatment frame
- it supports management, slower progression, and better quality/life expectancy language
- it is the most reusable branch for ordinary compiled write-back

### Branch 2: Selective Targeted Frontier

Main anchors:

- `src-hcm-011`
- `src-hcm-020`

Current meaning:

- this branch carries the strongest mechanism-linked ambition
- it matters because it shows HCM treatment is not only supportive
- it still sits below routine hierarchy because acute physiologic success and durable clinical ordering are not the same thing

### Branch 3: Older Conventional Therapy

Main anchor:

- `src-hcm-014`

Current meaning:

- this branch should stay visible as historical treatment reality
- it should not be merged with the newer frontier branch
- routine legacy use is not evidence-equivalent to modern hierarchy confidence

### Branch 4: Evidence-Skepticism Branch

Main anchor:

- `src-hcm-015`

Current meaning:

- this is not just rhetorical caution
- it is the branch that stops prescribing behavior from being misread as strong proof
- it should remain visible whenever the other three branches are summarized

## What This Comparison Clarifies

- `bounded management` is currently the safest top-layer treatment frame
- `targeted frontier` is the strongest innovation branch, but not the routine leader
- `older conventional therapy` is historical branch structure, not modern treatment confidence
- `evidence skepticism` is not anti-treatment; it is the control branch that keeps the layer honest

## What The Module Can Already Say

- HCM treatment should be read as four non-equivalent branches, not one ladder
- the ordinary compiled default should still lead with bounded management
- targeted frontier belongs in the module because it changes branch architecture
- evidence skepticism should remain visible whenever treatment language gets stronger

## What The Module Should Not Yet Say

- there is still no safe final intervention ranking
- targeted frontier should not be promoted above routine management on current evidence thickness alone
- older conventional therapy should not be upgraded into a modern high-confidence branch
- evidence skepticism should not erase the fact that real treatment structure exists
