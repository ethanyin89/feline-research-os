# FCV Mechanism-Control Memo

- Date: `2026-04-24`
- Scope: `src-fcv-001`, `src-fcv-002`, `src-fcv-007`, `src-fcv-015`, `src-fcv-016`, `src-fcv-017`, `src-fcv-023`

This memo exists to stop FCV mechanism from sounding like:

`a variable respiratory virus`

That is too compressed to control downstream writing.

## Core Takeaway

`the current FCV mechanism-control spine is diversity -> receptor/tissue reach -> persistence -> pathotype diversification, and that same spine explains why FCV control remains difficult`

## Current Mechanism-Control Layers

### Layer 1: Diversity And Evolution

This layer is mainly carried by:

- `src-fcv-001`
- `src-fcv-002`
- `src-fcv-015`
- `src-fcv-023`

Current meaning:

- FCV is a highly variable RNA virus with continuing genetic and antigenic diversification
- sequence variation is part of the control problem, not just a taxonomy detail
- one modern shell-review anchor plus one deep-extracted molecular bridge now hold this layer together

### Layer 2: Receptor And Tissue Reach

This layer is mainly carried by:

- `src-fcv-015`
- `src-fcv-016`

Current meaning:

- receptor biology and tissue distribution help explain broader lesion diversity
- FCV should not be modeled as only an upper-airway virus
- receptor/tropism logic now sits under a deeper mechanism spine rather than only under extension papers

### Layer 3: Persistence

This layer is mainly carried by:

- `src-fcv-002`
- `src-fcv-007`
- `src-fcv-017`

Current meaning:

- persistence and chronic carriage are built into the FCV system model
- long-term transmission pressure is partly maintained by carrier-state biology

### Layer 4: Pathotype Diversification And Control Consequence

This layer is mainly carried by:

- `src-fcv-001`
- `src-fcv-007`

Current meaning:

- FCV can generate multiple clinical forms, including more severe systemic disease
- one control strategy has to account for heterogeneity in both virus and phenotype

## What The Module Can Say Safely

- FCV is best modeled as a variable, persistence-capable, multi-phenotype viral system
- one deep-extracted mechanism bridge now supports that architecture directly
- one modern shell-review anchor now supports keeping control architecture and recognition caution inside the broader FCV frame
- receptor/tissue-tropism logic matters for understanding lesion diversity
- control difficulty is mechanistically connected to diversity and persistence
- mechanism writing should explain control problems, not float separately from them

## What The Module Should Not Say Yet

- do not reduce FCV to a simple respiratory-virus description
- do not imply receptor distribution alone explains virulence
- do not treat persistence as synonymous with vaccine failure
- do not pretend the current source pack has a full strain-by-strain virulence map
