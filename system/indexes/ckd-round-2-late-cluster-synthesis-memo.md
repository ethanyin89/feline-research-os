# CKD Round 2 Late-Cluster Synthesis Memo

Date: 2026-04-08  
Scope: `src-ckd-018`, `src-ckd-015`, `src-ckd-019`, `src-ckd-020`, `src-ckd-021`, `src-ckd-023`, `src-ckd-014`

This memo compresses the later Round 2 densification work after the first Tier A cluster.

## What This Cluster Changed

This cluster did not alter the main CKD backbone.

It made the vault more realistic and more layered in seven specific ways:

1. early detection now has a real metabolomic frontier
2. CKD-MBD is broader than phosphorus alone
3. hyperthyroidism is now an explicit interpretation confounder
4. ultrasound is now a bounded workup-context layer
5. aldosterone/MR signaling is now a second-generation target branch
6. senescence is now a direct feline aging-mechanism enrichment branch
7. practice friction is now an explicit translation constraint

## Paper-By-Paper Value

### `src-ckd-018`

This paper changed the early-detection branch most directly.

- the best single biomarker in this dataset was `S/U 3-hydroxykynurenine`
- the best overall performance came from multi-marker modelling
- SDMA did not dominate this dataset
- the result was still bounded by CKD2-only framing and no GFR/ultrasound adjudication

Net effect:

- the vault can now say `serial surveillance remains the backbone, but metabolomic panels may become an augmentation layer`

### `src-ckd-015`

This review widened the mineral branch.

- feline CKD-MBD is not only a phosphorus/PTH story
- hypercalcaemia risk matters
- FGF23 rises before azotaemic CKD
- phosphate restriction may interact with hypercalcaemia risk
- calcification remains mechanistically important but clinically underdefined

Net effect:

- `phosphorus remains core`, but it now sits inside a wider CKD-MBD network

### `src-ckd-019`

This review made concurrent disease interpretation more realistic.

- hyperthyroidism can mask CKD, and vice versa
- restoration of euthyroidism is a renal reassessment checkpoint
- iatrogenic hypothyroidism increases azotaemia risk
- symptom overlap reduces first-pass specificity in older cats

Net effect:

- hyperthyroidism is now an `interpretation confounder`, not just a generic comorbidity

### `src-ckd-020`

This review clarified the role of imaging.

- ultrasound is the reference imaging modality for the feline kidney
- it is useful for structure and differential assessment
- diffuse disease is harder to characterize than focal disease
- it should not be overpromoted as a standalone CKD answer

Net effect:

- ultrasound now belongs clearly in the `workup context` layer

### `src-ckd-021`

This review broadened the mediator-to-therapy branch.

- aldosterone/MR signaling is a plausible renal-injury pathway
- the branch includes vasculopathy, oxidative stress, inflammation, and profibrotic signaling
- MR antagonists are biologically interesting
- support is much stronger in model systems and human CKD than in feline efficacy evidence

Net effect:

- aldosterone/MR is now a `credible watchlist target branch`, not a feline standard-treatment tier

### `src-ckd-023`

This study enriched the aged-cat mechanism story.

- naturally occurring feline CKD is associated with renal telomere shortening
- renal senescence markers are increased
- the signal is kidney-focused rather than generalized across tissues

Net effect:

- senescence now has direct feline support as a `mechanism-enrichment branch`, but still does not replace fibrosis as the main backbone

### `src-ckd-014`

This study made implementation friction visible.

- guideline awareness was high
- systematic SBP measurement was still low
- renal diet recommendation was common, but adherence and rollout were imperfect
- owner constraints limited follow-up cadence

Net effect:

- the vault now has a real source for `implementation gap` and `common use is not evidence`

## Cross-Cluster Conclusions

### 1. The vault is now more realistic, not just richer

This cluster added realism in three directions at once:

- biologic realism
- workup realism
- implementation realism

### 2. Several new branches are now valid, but still secondary

These branches now deserve explicit mention:

- metabolomic early-detection augmentation
- CKD-MBD beyond phosphorus alone
- hyperthyroidism-confounded renal interpretation
- ultrasound workup context
- aldosterone/MR target logic
- senescence-associated aging biology
- implementation friction

But none of them displace:

- fibrosis-centered backbone
- core endpoint shortlist
- serial surveillance early-detection framing

### 3. The biggest remaining weakness is still treatment ranking

This cluster improved realism and branch coverage.

It did not solve the main unresolved translational question:

- which interventions have the strongest feline primary-study support
