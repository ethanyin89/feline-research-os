# CKD Disease-Modification Boundary Memo

- Date: `2026-04-10`
- Scope: `src-ckd-003`, `src-ckd-006`, `src-ckd-007`, `src-ckd-009`, `src-ckd-010`, `src-ckd-013`, `src-ckd-014`, `src-ckd-021`

This memo exists because the current CKD translation layer is repeatedly being forced to answer a narrower question:

`where does management and progression control stop, and where would true disease-modification language begin?`

## Core Takeaway

`the current CKD treatment layer is clearer at the disease-modification boundary than at any final product-specific hierarchy`

## Current Boundary Layers

### Layer 1: Stronger Management And Progression-Control Zone

Main anchors:

- `src-ckd-003`
- `src-ckd-006`
- `src-ckd-009`
- `src-ckd-010`

Current meaning:

- renal diet, phosphorus control, blood-pressure management, and proteinuria-aware management all belong here
- this zone is the strongest current practical treatment territory in the vault
- it supports progression-burden influence more clearly than structural reversal

### Layer 2: Supportive And Implementation-Realism Zone

Main anchors:

- `src-ckd-007`
- `src-ckd-014`

Current meaning:

- common clinical use and field practice sit here
- this zone matters because execution friction changes real-world translation
- it also exposes where routine use is running ahead of strong efficacy evidence

### Layer 3: Future-Oriented Or Second-Generation Translational Zone

Main anchors:

- `src-ckd-021`
- `src-ckd-013`

Current meaning:

- MR antagonism and broader trial-outcome logic belong here
- this zone matters because it sharpens what a stronger future CKD intervention claim would need to look like
- it is not yet the same thing as validated feline disease modification

## What This Boundary Clarifies

- the current vault is strongest for management and progression control
- the vault can support influence-on-burden language better than structural-reversal language
- implementation realism is part of translation, but not proof of disease modification
- future-oriented plausibility belongs in the module, but below validated feline efficacy

## What The Module Can Already Say

- current CKD translation is real and practically important
- `renal diet` remains the clearest baseline-supported intervention branch
- phosphorus control, proteinuria, and blood-pressure management can be defended as progression-linked treatment logic
- CKD treatment language should more often say `management`, `progression control`, or `complication burden reduction` than `disease modification`

## What The Module Should Not Yet Say

- there is still no safe strong claim of direct structural disease reversal
- common adjunct use should not be promoted into disease-modifying proof
- future-oriented branches should not be rewritten as validated feline clinical tiers
- earlier recognition should not be misread as proven prevention
