# Diabetes SGLT2 Safety Regulatory Boundary Memo

- Date: `2026-04-24`
- Scope: `src-diabetes-011`, `src-reg-010`, `src-reg-011`, with comparator treatment context from `src-diabetes-024`

This memo exists because SGLT2 inhibitors are both a treatment branch and a regulatory-claim trap.

## Core Takeaway

`SGLT2 inhibitors are a real feline diabetes treatment branch with primary U.S. FDA source cards now ingested, but the branch must keep candidate selection and euglycemic DKA risk ahead of convenience or route enthusiasm`

## Current Treatment Signal

Main source:

- [src-diabetes-011 deep extraction round 1](src-diabetes-011-deep-extraction-round1.md)

Current compiled evidence supports:

- bexagliflozin and velagliflozin are named as recently FDA approved for uncomplicated feline diabetes
- suitable candidates need residual beta-cell function
- serious complications include euglycemic diabetic ketoacidosis

## Safety Boundary

The branch is not:

`oral route = easier diabetes management`

The branch is:

`candidate selection + residual beta-cell function + monitoring + euglycemic DKA risk`

That distinction should govern every translation or regulatory page.

## Primary FDA Source Boundary

`src-diabetes-011` supported the original regulatory flag.

That flag now has primary FDA source support:

- [src-reg-010 Bexacat FDA FOI](../../raw/regulations/src-reg-010-bexacat-fda-foi.md)
- [src-reg-011 Senvelgo FDA FOI](../../raw/regulations/src-reg-011-senvelgo-fda-foi.md)
- [diabetes SGLT2 primary FDA source memo](diabetes-sglt2-primary-fda-source-memo.md)

The regulatory brief should no longer say no primary SGLT2 product source exists.

It should instead say:

`U.S. primary SGLT2 product sources exist, but current label wording, contraindications, monitoring instructions, and non-U.S. jurisdictions still need explicit source control`

## Comparator Boundary

`src-diabetes-024` supports glargine U300 as a promising insulin branch, but it is not a direct comparator to SGLT2 treatment.

So the current treatment map should not rank:

- SGLT2 inhibitor versus insulin
- oral route versus injectable route
- FDA approval versus clinical superiority

without primary label and full-text clinical evidence.

## What The Module Can Say Now

- SGLT2 inhibitors belong in the diabetes translation and regulatory-boundary pages
- uncomplicated diabetes and residual beta-cell function are key candidate-selection concepts
- euglycemic DKA is a must-track safety endpoint
- Bexacat and Senvelgo have primary FDA FOI source cards in the vault

## What The Module Should Not Say Yet

- do not infer use in complicated diabetes
- do not replace insulin/diet framing wholesale
- do not turn U.S. FOI summaries into global regulatory claims
- do not rank products by convenience or route-fit
