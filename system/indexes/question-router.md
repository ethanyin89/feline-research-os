# Question Router

这页是跨病种入口。

If you already know the kind of question you have, start here before choosing the disease page.

If you do not yet know your question shape, go back to:

- [ask first](../../start.md)
- [reader start here](reader-start-here.md)

If you want the strongest answer surface map first, use:

- [best answer surfaces](best-answer-surfaces.md)

## Fastest Cross-Disease Moves / 最快跨病种动作

如果你现在不是在找"哪个栏目"，而是在找"哪种答案面"，先用这 5 条：

- `What is X?` / `什么是X？` / `解释一下X`
  Open the disease `what-is-*` page (or dashboard if not yet compiled)
- `I want the fastest disease answer`
  Open the disease `current-state-dashboard`
- `I want the best structured overview`
  Open the disease `synthesis-index`
- `I want to verify a claim`
  Open [verify-a-claim](verify-a-claim.md)
- `I want this question to turn into write-back if it matters repeatedly`
  Open [query-to-write-back](query-to-writeback.md)
- `I want the strongest answer surface map first`
  Open [best-answer-surfaces](best-answer-surfaces.md)

如果这些还不够，再按问题层级往下选。
