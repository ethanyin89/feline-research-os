# HCM Treatment Evidence Memo

- Date: `2026-04-10`
- Scope: `src-hcm-008`, `src-hcm-011`, `src-hcm-014`, `src-hcm-015`, `src-hcm-020`

This memo exists to stop the current feline HCM treatment branch from collapsing into one vague sentence like:

`there are treatments for HCM`

That sentence is directionally true, but it is too flat for this vault.

## Core Takeaway

`the feline HCM treatment branch is real, but it is structurally split between bounded management, selective targeted frontier logic, and explicit evidence skepticism`

## Current Treatment Layers

### Layer 1: Bounded Management Anchor

This layer is mainly carried by:

- `src-hcm-008`

Current meaning:

- the safest modern treatment framing is still management-oriented
- the review supports control of clinical signs, slower progression, and better quality and length of life
- this is stronger than pure symptom relief language, but weaker than disease-reversal language

### Layer 2: Targeted Mechanism-Linked Frontier

This layer is mainly carried by:

- `src-hcm-011`
- `src-hcm-020`

Current meaning:

- the translational branch has real mechanism-linked ambition
- targeted contractility modulation and remodeling biology are valid frontier branches
- they should remain below routine treatment hierarchy until broader evidence is better bounded

### Layer 3: Older Conventional-Therapy Branch

This layer is mainly carried by:

- `src-hcm-014`

Current meaning:

- older conventional therapy belongs in the treatment map
- it should not be silently merged with newer targeted logic
- routine historical use and strong modern hierarchy are different things

### Layer 4: Evidence-Skepticism Anchor

This layer is mainly carried by:

- `src-hcm-015`

Current meaning:

- treatment evidence quality is not a side note; it is one of the main branches of the HCM treatment layer
- what clinicians prescribe and what evidence strongly supports are not the same thing
- the branch must remain overclaim-sensitive even when practice patterns look stable

## What The Module Can Already Say

- HCM treatment should not be modeled as a flat intervention list
- current safest language is `bounded management + selective targeted frontier`
- mechanism-linked translational work belongs in the module, but below routine hierarchy certainty
- evidence skepticism is a core structural feature of the HCM treatment branch, not a footnote

## What The Module Should Not Yet Say

- there is no stable final intervention ranking yet
- there is no safe one-line claim that HCM treatment is strongly evidence-settled
- targeted frontier papers should not be promoted into routine-ready hierarchy
- conventional historical therapy should not be merged with modern targeted optimism
