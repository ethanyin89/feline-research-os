# IBD-Lymphoma Boundary Memo

## Core Takeaway

`the IBD-versus-small-cell-lymphoma problem is multimodal, not a one-test question`

## Current Boundary Architecture

- `diagnosis of exclusion` is the top-level frame
- `clinical activity` helps stage burden but does not classify disease identity
- `imaging` can tilt suspicion, especially when muscularis thickening is present
- `biopsy site selection` can materially change whether lymphoma is found
- `histopathology-report structure` can stabilize interpretation language, but it does not make one-site tissue complete
- `microbiota and inflammatory-signaling differences` deepen explanation and may support future stratification

## Practical Boundary Sequence

1. **Do not start at "IBD".** Start at chronic enteropathy with mimics live.
2. **Let imaging change the pre-test pressure.** Muscularis thickening is the strongest current ultrasound pressure toward lymphoma; lymphadenopathy is abnormality-supporting but less discriminative.
3. **Let that pressure shape tissue strategy.** If lymphoma remains a live concern, duodenal-only tissue is weaker than paired duodenal / ileal evaluation for exclusion.
4. **Use pathology as the central gate, but not as a one-site shortcut.** A nonlymphoma result from one intestinal site should be read against imaging pressure and sampling completeness. Structured report classification belongs here as workflow support, not as a substitute for sampling strategy.
5. **Use markers as tiered support.** Bcl-2 is a lymphoma-leaning tissue marker with overlap; fecal S100A12 is disease-versus-health support; metabolomics is the strongest current frontier separation signal but not a routine workup leader.

## What The Module Can Already Say

- chronic enteropathy activity and response should not be mistaken for inflammatory-disease identity
- older cats with muscularis thickening should raise stronger lymphoma concern than cats without that pattern
- ileal evaluation can matter enough to change whether small-cell lymphoma is detected at all
- microbiota differences between IBD and small-cell lymphoma are real enough to support a biologic boundary branch
- direct feline lymphoma-side material now helps keep the malignant side of the boundary from being under-modeled
- histopathology-report classification adds a real workflow layer below pathology, but the current source read supports language normalization more strongly than final disease-class certainty
- Bcl-2 adds a stronger lymphoma-leaning tissue-marker layer than shared inflammatory markers, but still overlaps too much for stand-alone use
- fecal S100A12 supports noninvasive inflammatory disease detection, but it does not separate IBD from lymphoma in the current source read
- metabolomics is the strongest current frontier-separation branch, but it should be treated as future stratification rather than routine diagnostic leadership

## Current Boundaries

- `muscularis thickening` is not the same thing as lymphoma diagnosis
- `one-site negative biopsy` is not the same thing as meaningful lymphoma exclusion
- `microbiota difference` is not the same thing as routine diagnostic readiness
- older lymphoma case-series material is useful for disease-form anchoring, but not for modern stand-alone diagnosis
- report-language normalization is not the same thing as resolving the IBD-versus-lymphoma boundary
- `marker signal` is not the same thing as practical workup replacement
