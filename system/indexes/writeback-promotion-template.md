# Write-Back Promotion Template

这页不是解释页。

这页是一个可以直接复制的短模板，用在：

- memo
- dashboard update
- synthesis update
- output write-back suggestion

## Short Template / 短模板

```md
## Promotion Judgment

- repeated? `yes / no`
- structurally clarifying? `yes / no`
- evidence-safe enough for this layer? `yes / no`
- smallest durable home: `topic update / memo / dashboard update / navigation update / bilingual page / stay chat-only`

### Reason

- what is repeating:
- what becomes clearer:
- what is still too thin, if anything:

### Decision

- promote / hold
```

## One-Line Use Rule / 一句话用法

如果你已经准备做 write-back，就先贴这个模板，再决定是 `promote` 还是 `hold`。
