# IBD Tissue-Marker Boundary Memo

## Core Takeaway

`tissue and molecular markers add boundary pressure, but none of them currently justify one-marker workup leadership`

## Current Marker Architecture

- `MDR1` looks more like a shared chronic-enteropathy disturbance than a clean separator
- `COX-2 mRNA` does not separate IBD from low-grade alimentary lymphoma in the current pilot data
- `epithelial COX-2 immunoexpression` looks like a shared epithelial response signal rather than a disease-class discriminator
- `Bcl-2` is more lymphoma-leaning than the COX-2 signals, but overlap remains high enough to require bounded interpretation

## What The Module Can Already Say

- there is real marker-level structure below histopathology in the boundary branch
- most current marker signals are better at adding biologic depth than replacing core workup layers
- Bcl-2 is the most lymphoma-leaning of the current trio, but it still does not justify marker-first compression

## Current Boundaries

- `shared abnormality` is not the same thing as `class separator`
- `statistical difference` is not the same thing as `operational diagnostic usefulness`
- `tissue-marker signal` is not the same thing as `one-marker diagnosis`
