# Endpoint Selection Guide: 按药物靶点选择评价终点

本指南根据药物作用机制推荐猫糖尿病模型的终点组合，供CRO药效试验方案设计参考。

---

## 快速决策矩阵

| 药物类型 | 核心机制终点 | 药效终点 | 安全性终点 | 特殊考量 |
|:---|:---|:---|:---|:---|
| **胰岛素增敏剂** | FSIVGTT (SI), EHC | CGM, 果糖胺 | GFR, 肝功能 | 需区分外周 vs 肝脏抵抗 |
| **促泌剂** | 胰岛素/C肽分泌曲线 | 果糖胺, CGM | β-HB, IAPP组织病理 | 促泌剂可能增加IAPP沉积风险 |
| **SGLT2 抑制剂** | 尿糖排泄 | CGM, 果糖胺 | β-HB（强化）, GFR, 电解质 | 酮症风险为首要安全关注 |
| **β细胞保护剂** | 胰岛组织病理, IAPP, β细胞量 | 果糖胺, 胰岛素分泌动态 | 常规安全性 | 猫是IAPP研究的独特模型 |
| **GLP-1 类似物** | 胰岛素分泌动态, 食欲/体重 | CGM, 果糖胺 | 胃肠道反应, 胰腺炎标志物 | 需长期随访 |

---

## 1. 胰岛素增敏剂 (Insulin Sensitizers)

**代表药物**：噻唑烷二酮类 (TZDs)、二甲双胍

### 机制验证终点

| 终点 | 方法 | 依据 |
|:---|:---|:---|
| **外周胰岛素敏感性** | insulin-modified FSIVGTT → SI | 量化骨骼肌葡萄糖处置能力 |
| **全身胰岛素敏感性** | EHC → GIR | 金标准，综合外周和脂肪组织 |
| **肝脏胰岛素敏感性** | EHC + 标记葡萄糖 | 若靶点涉及肝糖输出抑制 |

### 药效终点

| 终点 | 时间窗 | 说明 |
|:---|:---|:---|
| **果糖胺** | 给药后 2-4 周 | 反映中期糖代谢改善 |
| **CGM** | 给药后 1-2 周 | 日常血糖轨迹变化 |
| **空腹血糖** | 全程 | 辅助指标，注意应激干扰 |

### 安全性终点

- GFR, 血清肌酐（肾功能）
- ALT, AST（肝功能，尤其TZDs）
- 体重（水肿风险）

### 设计要点

1. **基线必须证明存在胰岛素抵抗**：入组前 FSIVGTT 或 EHC 确认 SI/GIR 低于正常
2. **区分急性 vs 稳定期**：新诊断糖尿病猫的胰岛素抵抗部分来自葡萄糖毒性，可随控糖改善
3. **FSIVGTT 无法拆分肝脏贡献**：若药物主要作用于肝糖异生，SI 可能低估真实药效

---

## 2. 促泌剂 (Secretagogues)

**代表药物**：磺脲类（格列吡嗪）、格列奈类

### 机制验证终点

| 终点 | 方法 | 依据 |
|:---|:---|:---|
| **胰岛素分泌能力** | IVGTT 胰岛素曲线 | 评估一相/二相分泌 |
| **C肽** | 空腹及刺激后 C肽 | 反映内源性胰岛素生成 |

### 药效终点

| 终点 | 时间窗 | 说明 |
|:---|:---|:---|
| **果糖胺** | 给药后 2-4 周 | 中期控糖效果 |
| **CGM** | 全程 | 监测低血糖风险 |

### 安全性终点（关键）

| 终点 | 依据 | 证据强度 |
|:---|:---|:---|
| **β-HB** | 促泌剂理论上降低酮症风险，但需监控 | - |
| **低血糖发生率** | CGM 记录 | - |
| **IAPP/胰岛淀粉样沉积** | 组织病理（终点研究） | [低] n=4 |

### 特殊警示

> **Henson 2006 数据**：格列吡嗪治疗的诱导糖尿病猫 18 月后 4/4 出现胰岛淀粉样沉积，而胰岛素治疗组仅 1/4。
>
> **含义**：促泌剂可能通过刺激 IAPP 与胰岛素共分泌，增加淀粉样沉积风险。
>
> **证据强度**：[低] 仅 n=4，需进一步验证。

### 设计要点

1. **长期研究应纳入胰岛组织病理**：评估 IAPP 沉积和 β 细胞量
2. **猫是研究促泌剂-IAPP 关联的独特模型**：啮齿类不自发形成胰岛淀粉样变

## 3. SGLT2 抑制剂

**代表药物**：达格列净、卡格列净、恩格列净

### 机制验证终点

| 终点 | 方法 | 依据 |
|:---|:---|:---|
| **尿糖排泄** | 24h 尿糖定量 | 直接反映药物机制 |
| **肾糖阈** | 血糖/尿糖关系曲线 | 评估阈值下移程度 |

### 药效终点

| 终点 | 时间窗 | 说明 |
|:---|:---|:---|
| **果糖胺** | 给药后 2-4 周 | 中期控糖效果 |
| **CGM** | 全程 | 日常血糖轨迹 |
| **体重** | 全程 | SGLT2i 常伴随体重下降 |

### 安全性终点（强化监测）

| 终点 | 频率 | 阈值/行动 |
|:---|:---|:---|
| **β-HB** | 基线, 24h, 72h, 每周 | > 2.0 mmol/L → 血气评估 |
| **血气** | 基线, 72h, β-HB 升高时 | pH, HCO₃⁻, 阴离子间隙 |
| **电解质** | 每周 | Na, K, Cl |
| **GFR/肌酐** | 基线, 每 2 周 | 监控肾功能变化 |
| **尿比重** | 每周 | 渗透性利尿评估 |

### 酮症风险管理决策树

```
β-HB 监测结果
    │
    ├─ < 1.0 mmol/L → 继续常规监测
    │
    ├─ 1.0-2.0 mmol/L → 增加监测频率至每日
    │
    ├─ 2.0-2.55 mmol/L → 立即血气评估
    │                    │
    │                    ├─ pH 正常, 电解质正常 → 密切观察, 考虑调整剂量
    │                    │
    │                    └─ pH 下降 或 阴离子间隙升高 → 停药, 治疗
    │
    └─ > 2.55 mmol/L → 停药, 完整DKA评估和治疗
```

### 特殊警示

> **掩蔽性酸中毒**：高 β-HB + 正常 pH 可能是低氯性代谢性碱中毒掩盖了酮症酸中毒。
>
> **不能仅凭 pH 正常排除危险**：必须结合 β-HB、阴离子间隙、chloride gap 综合判断。

## 4. β细胞保护剂 / IAPP 抑制剂

**代表药物**：IAPP 聚集抑制剂、抗凋亡药物、GLP-1 类似物（部分）

### 机制验证终点（核心）

| 终点 | 方法 | 依据 |
|:---|:---|:---|
| **胰岛淀粉样沉积** | 组织病理 + 刚果红染色 + 免疫组化 | 金标准，需终点活检/尸检 |
| **β细胞量** | 胰岛形态计量 | 与人 T2DM 类似（猫约减少 50%） |
| **IAPP 表达/分布** | 免疫组化、Western blot | 评估 IAPP 聚集状态 |

### 药效终点

| 终点 | 时间窗 | 说明 |
|:---|:---|:---|
| **胰岛素分泌动态** | IVGTT 曲线 | 评估 β 细胞功能保护 |
| **果糖胺** | 长期随访 | 控糖状态维持 |
| **糖尿病进展/缓解率** | 终点 | 临床结局 |

### 设计要点

1. **猫是 IAPP 研究的不可替代模型**：
   - 猫和人 IAPP 20-29 位区域具有 β-sheet 形成倾向
   - 普通啮齿类不具备此特性，不自发形成胰岛淀粉样变
   - 猫 > 80% 糖尿病个体有胰岛淀粉样沉积（人 > 90%）

2. **需要长期随访**：β 细胞保护效果需要数月至数年观察

3. **组织学终点需要活检或尸检**：血浆 IAPP 测定在猫中验证有限

## 5. GLP-1 受体激动剂

**代表药物**：利拉鲁肽、司美格鲁肽、艾塞那肽

### 机制验证终点

| 终点 | 方法 | 依据 |
|:---|:---|:---|
| **葡萄糖依赖性胰岛素分泌** | IVGTT + GLP-1 激动剂 | 增强效应 |
| **食欲/摄食量** | 每日记录 | GLP-1 抑制食欲 |
| **胃排空** | 放射性标记餐 | 延迟胃排空 |

### 药效终点

| 终点 | 时间窗 | 说明 |
|:---|:---|:---|
| **体重** | 全程 | 常见的次要效应 |
| **果糖胺** | 给药后 2-4 周 | 控糖效果 |
| **CGM** | 全程 | 餐后血糖变化 |

### 安全性终点

| 终点 | 说明 |
|:---|:---|
| **胃肠道反应** | 呕吐、腹泻、食欲废绝 |
| **胰腺炎标志物** | 猫特异性胰脂肪酶 (fPLI) |
| **肾功能** | GFR, 肌酐 |

## 6. 神经/视网膜并发症干预

**适用药物**：醛糖还原酶抑制剂、抗氧化剂、神经营养因子

### 神经病变终点

| 终点 | 方法 | 依据 |
|:---|:---|:---|
| **神经传导速度** | 电生理 | 功能评估 |
| **跖行评分** | 临床评分 | 临床表现 |
| **神经组织病理** | 活检/尸检 | 施旺细胞损伤 |

### 视网膜病变终点

| 终点 | 方法 | 依据 |
|:---|:---|:---|
| **眼底照相** | 数字眼底镜 | 微动脉瘤、出血 |
| **荧光素血管造影** | 造影 | 毛细血管无灌注 |

### 设计要点

- **需要长期随访**：慢性并发症需数年发展
- **猫的优势**：成年猫不易发生白内障，有利于眼底观察
- **猫神经病变特点**：主要积累果糖而非山梨醇

## 通用设计原则

### 1. 基线确认

- 所有机制研究须在入组时确认模型状态（胰岛素抵抗程度、血糖水平、酮体状态）
- 新诊断糖尿病猫的部分代谢异常可能来自葡萄糖/脂毒性，随控糖改善

### 2. 分层变量控制

| 变量 | 影响 | 控制方法 |
|:---|:---|:---|
| **年龄** | 独立影响胰岛素敏感性 | 年龄匹配或分层分析 |
| **品种** | 缅甸猫糖尿病风险 4× | 记录并分析 |
| **饮食** | 自由采食量影响代谢 | 标准化喂养方案 |
| **体况** | 肥胖 vs 非肥胖 | BCS 分层 |

### 3. 模型边界声明

**猫糖尿病模型适用于**：
- IAPP/胰岛淀粉样变研究
- 外周胰岛素增敏评价
- 糖尿病神经病变和视网膜病变研究
- 遗传易感性研究（缅甸猫）

**猫糖尿病模型不适用于**：
- 动脉粥样硬化终点
- 临床高血压终点
- 典型糖尿病性肾病 (DKD) 终点
