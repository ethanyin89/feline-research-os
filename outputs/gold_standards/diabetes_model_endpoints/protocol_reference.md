# Protocol Reference Card: Feline Diabetes Model Key Endpoints

本文档提供猫糖尿病模型关键评价终点的最小可行协议参数，供CRO药效试验方案设计参考。

---

## 1. Insulin-Modified FSIVGTT (胰岛素修饰频繁采样静脉葡萄糖耐量试验)

**目的**：量化外周组织（主要为骨骼肌）胰岛素敏感性，输出 SI (Insulin Sensitivity Index)

### 协议参数 (Feldhahn 1999)

| 参数 | 规格 |
|:---|:---|
| **禁食** | 12小时 |
| **麻醉** | 丙泊酚 5 mg/kg IV（置管用） |
| **血管通路** | 颈静脉导管 18G × 8 cm |
| **葡萄糖剂量** | 0.3 g/kg IV，30-45秒推注 |
| **胰岛素剂量（正常猫）** | 0.05 U/kg 人常规胰岛素 |
| **胰岛素剂量（糖尿病猫）** | 0.3 U/kg 人常规胰岛素 |
| **胰岛素注射时间** | 葡萄糖注射后 20 min |
| **采样时间点** | 基线 (-15, -10, -5, -1 min) + 葡萄糖后 27 个时间点，持续 3 小时 |
| **葡萄糖检测** | YSI glucose analyser 或同等 |
| **胰岛素检测** | Pharmacia insulin RIA kit（可检测人和猫胰岛素）或同等 |
| **建模软件** | MINMOD (R.N. Bergman) |

### 参考值

| 指标 | 正常猫 | 糖尿病猫 | 来源 |
|:---|:---|:---|:---|
| SI | 3.22 × 10⁻⁴ min/μU/ml (范围 1.71-5.23) | 0.58 × 10⁻⁴ min/μU/ml (范围 0.14-0.88) | Feldhahn 1999 [n=4] |

### 方法学边界

- **主要测量外周葡萄糖处置**（骨骼肌），无法量化肝脏胰岛素抵抗
- 若需评估肝糖输出，须结合 EHC + 标记葡萄糖输注
- 跨物种绝对值比较无效，应比较同物种内正常/疾病状态的相对变化
- **证据强度**：[低] 基于 n=4 最终分析

## 2. EHC (正常血糖高胰岛素钳夹)

**目的**：金标准胰岛素敏感性评估，测量在恒定高胰岛素状态下维持正常血糖所需的葡萄糖输注率

### 协议原理 (Hoenig 2012)

| 参数 | 说明 |
|:---|:---|
| **原理** | 静脉输注人重组胰岛素 → 维持恒定高胰岛素状态 → 调整葡萄糖输注率以维持正常血糖 |
| **主要输出** | 葡萄糖输注率 (GIR, mg/kg/min) |
| **解释** | GIR 高 = 胰岛素敏感；GIR 低 = 胰岛素抵抗 |
| **优势** | 不依赖猫特异性胰岛素检测（输注的是人胰岛素） |

### 适用场景

- 验证胰岛素增敏剂药效
- 定量评估外周和脂肪组织胰岛素敏感性
- 研究饮食干预对胰岛素敏感性的影响

### 方法学边界

- 操作复杂度高于FSIVGTT，需持续血糖监测和葡萄糖输注调整
- 可结合标记葡萄糖技术测定肝糖输出

## 3. β-HB 床旁监测 (酮症安全性评价)

**目的**：快速评估酮体状态，用于 SGLT2 抑制剂等易诱发酮症药物的安全性监控

### 设备与参数 (Zeugswetter 2012)

| 参数 | 规格 |
|:---|:---|
| **设备** | Precision Xtra Blood Glucose & Ketone Monitoring System（或同等验证设备） |
| **样本** | 全血 ~1.5 μL |
| **测量范围** | 0-8.0 mmol/L |
| **与实验室相关性** | r = 0.97, P < 0.001 |

### 临床阈值与解释

| β-HB 水平 | 临床解释 | 行动 |
|:---|:---|:---|
| **< 2.55 mmol/L** | 酮症酸血症可能性低 | 可排除严重DKA，考虑其他鉴别诊断 |
| **> 2.55 mmol/L** | 需警惕酮症酸血症 | 立即进行血气、电解质评估 |

### 诊断效能

| 指标 | 数值 | 来源 |
|:---|:---|:---|
| 敏感性 | 94% | Zeugswetter 2012 |
| 特异性 | 68% | Zeugswetter 2012 |
| AUC | 0.84 (95% CI: 0.787-0.859) | Zeugswetter 2012 |

### 关键陷阱

1. **高浓度负偏倚**：β-HB 很高时，POC设备可能低估真实值
2. **掩蔽性酸中毒**：高β-HB + 正常pH 可能是混合性酸碱紊乱（低氯性代谢性碱中毒掩盖酮症酸中毒）
3. **阈值用于排除而非确诊**：> 2.55 mmol/L 需进一步血气确认，不能单独确诊DKA

### 监测建议（SGLT2抑制剂等高风险药物）

| 时间点 | 监测项目 |
|:---|:---|
| 基线 | β-HB, 血气, 电解质 |
| 给药后 24h | β-HB |
| 给药后 72h | β-HB, 血气 |
| 每周 | β-HB |
| β-HB > 2.0 mmol/L 时 | 立即血气、电解质、临床评估 |

## 4. 果糖胺 (中短期糖代谢监测)

**目的**：评估过去 7-10 天的平均血糖状态，用于诊断确认和治疗效果监测

### 参数 (Thoresen 1996)

| 参数 | 说明 |
|:---|:---|
| **反映周期** | 7-10 天平均血糖 |
| **主要价值** | 区分应激性高血糖 vs 真实糖尿病 |
| **采样** | 静脉血清 |

### 临床应用

| 场景 | 应用方式 |
|:---|:---|
| **入组筛选** | 果糖胺升高 + 持续高血糖 + 临床症状 → 确证糖尿病 |
| **药效评估** | 连续监测果糖胺趋势（下降 = 控糖改善） |
| **排除应激** | 单次高血糖但果糖胺正常 → 可能为应激性高血糖 |

### 局限性

- 受血清蛋白浓度影响（低蛋白血症 → 假性降低）
- 受甲状腺功能影响
- **无法识别短暂低血糖或高血糖波动**
- 不适合用于微调胰岛素剂量

## 5. 连续血糖监测 (CGM)

**目的**：评估日常真实血糖轨迹，优于单次采血或药理性糖负荷测试

### 原理 (Hoenig 2012)

- 7天连续监测显示：多数长期肥胖、胰岛素抵抗猫的日常血糖与瘦猫无差异
- 只有真正进入糖尿病前期的猫（~1/8）显示日常血糖整体抬高约25%
- **IVGTT/OGTT 不能可靠预测日常血糖稳态**

### 适用场景

- 评估真实日常血糖控制
- 识别短暂低血糖风险
- 区分"代谢测试异常"与"日常血糖异常"

## 6. 肾脏安全性监测

**目的**：监控药物肾毒性（猫本身极少并发典型DKD）

### 推荐指标 (Paepe 2015)

| 指标 | 说明 |
|:---|:---|
| **GFR** | 金标准，碘海醇清除率 |
| **血清肌酐** | 常规筛查 |
| **UPC** | 尿蛋白/肌酐比，> 0.4 需关注 |
| **UAC** | 尿白蛋白/肌酐比，用于纵向追踪 |

### 关键判断原则

> 糖尿病猫基线 GFR 和肌酐通常与健康猫无显著差异。
> **若给药后出现 GFR 显著下降 → 首先怀疑药物肾毒性，而非糖尿病自然病程。**
