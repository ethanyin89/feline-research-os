# Core Disease Frame

- HCM should currently be modeled as a:
  - structure-first cardiomyopathy module
- The strongest current top-line architecture is:
  - structural phenotype confirmation
  - bounded screening augmentation
  - stratified genotype pressure
  - remodeling beneath visible phenotype
  - treatment reality constrained by evidence skepticism

Bottom line:

Do not model HCM as a biomarker-first or mutation-only disease.

---

# What Leads Recognition

- echocardiography
- gross morphometry
- phenotype definition before biomarker enthusiasm

Internal implication:

Structural confirmation should stay above screening augmentation.

---

# Endpoint Hierarchy

## Current order

- structural confirmation
- screening augmentation
- injury / severity pressure
- pathology-depth context

## Marker rule

- troponin I = injury / severity pressure
- NT-proBNP = severe-disease flag

---

# Genotype-Severity Logic

- genetics matters
- but not as a binary mutation present / absent story
- what currently matters most:
  - mutation dosage
  - age-related penetrance

Bottom line:

Genotype should deepen phenotype interpretation, not replace it.

---

# Remodeling Depth

- remodeling is not a passive after-effect
- fibrosis and late-stage change deepen the phenotype map
- end-stage HCM should be read as a remodeled phenotype

Bottom line:

Do not flatten late disease into “just more hypertrophy.”

---

# Treatment Boundary

## Safest current frame

- bounded management
- selective targeted frontier

## What must stay visible

- prescribing behavior
- evidence skepticism

Bottom line:

The treatment branch is real, but flat treatment ranking is still too strong.

---

# What We Can Say Now

## Defensible now

- structure-first recognition
- endpoint hierarchy is real
- genotype pressure is meaningful
- remodeling deepens phenotype
- treatment skepticism is structurally important

## Not defensible yet

- one-marker diagnosis
- mutation-only phenotype prediction
- final treatment ranking
- route-level regulatory recommendation
