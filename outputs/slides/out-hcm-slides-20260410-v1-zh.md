# 核心疾病框架

- HCM 当前应被建模成：
  - structure-first cardiomyopathy module
- 当前最强的顶层架构是：
  - structural phenotype confirmation
  - bounded screening augmentation
  - stratified genotype pressure
  - remodeling beneath visible phenotype
  - treatment reality constrained by evidence skepticism

Bottom line:

不要把 HCM 建模成 biomarker-first 或 mutation-only disease。

---

# 什么主导识别

- echocardiography
- gross morphometry
- phenotype definition before biomarker enthusiasm

内部含义：

Structural confirmation 应继续放在 screening augmentation 之上。

---

# Endpoint hierarchy

## 当前顺序

- structural confirmation
- screening augmentation
- injury / severity pressure
- pathology-depth context

## Marker rule

- troponin I = injury / severity pressure
- NT-proBNP = severe-disease flag

---

# Genotype-Severity 逻辑

- genetics 很重要
- 但不是一个 binary mutation present / absent story
- 当前最重要的是：
  - mutation dosage
  - age-related penetrance

Bottom line:

Genotype 应加深 phenotype interpretation，而不是替代它。

---

# Remodeling depth

- remodeling 不是被动后果
- fibrosis 和 late-stage change 会加深 phenotype map
- end-stage HCM 应被读成 remodeled phenotype

Bottom line:

不要把晚期病程压成“只是更重的 hypertrophy”。

---

# Treatment boundary

## 当前最安全的框架

- bounded management
- selective targeted frontier

## 必须继续保留

- prescribing behavior
- evidence skepticism

Bottom line:

Treatment branch 是真实存在的，但 flat treatment ranking 仍然太强。
