# Mechanism Backbone

- Current strongest disease backbone: `renal fibrosis / tubulointerstitial fibrosis`
- Primary-study support links fibrosis to:
  - azotemia
  - hyperphosphatemia
  - anaemia
- Important bridge variables:
  - proteinuria
  - phosphorus
  - systolic blood pressure

Bottom line:

CKD should not be modeled as a loose symptom cluster.  
It is a progressive structural disease with measurable downstream signals.

Important context endpoints:

- PTH

---

# Endpoint Logic

The endpoint system should be split into three buckets:

1. Diagnosis and staging
   - creatinine
   - USG
   - SDMA

2. Monitoring and prognosis
   - UPCR or proteinuria
   - systolic blood pressure
   - phosphorus

3. Pathology-linked context
   - anaemia
   - PTH
   - potassium

This matters because some variables are clinically easy to use, while others are more mechanistically informative.
