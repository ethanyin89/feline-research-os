# Core Disease Frame

- HCM should currently be modeled as a:
  - structure-first cardiomyopathy module
- The strongest current top-line architecture is:
  - structural phenotype confirmation
  - bounded screening augmentation
  - stratified genotype pressure
  - remodeling beneath visible phenotype
  - treatment reality constrained by evidence skepticism

Bottom line:

Do not model HCM as a biomarker-first or mutation-only disease.

# Genotype-Severity Logic

- genetics matters
- but not as a binary mutation present / absent story
- what currently matters most:
  - mutation dosage
  - age-related penetrance

Bottom line:

Genotype should deepen phenotype interpretation, not replace it.

---

# Remodeling Depth

- remodeling is not a passive after-effect
- fibrosis and late-stage change deepen the phenotype map
- end-stage HCM should be read as a remodeled phenotype

Bottom line:

Do not flatten late disease into “just more hypertrophy.”

---

# Treatment Boundary

- selective targeted frontier

## What must stay visible

- prescribing behavior
- evidence skepticism

Bottom line:

The treatment branch is real, but flat treatment ranking is still too strong.
